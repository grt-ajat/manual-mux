Forward clutch assembly inspection (TB-50LS)

1. Forward clutch return spring sub-assembly measurement

1. Measure the forward clutch return spring sub-assembly using a vernier caliper.

Note

  • Using vernier calipers, measure the free length of the forward clutch return spring sub-assembly.
  • If the free length is smaller than the minimum, replace the forward clutch return spring sub-assembly.

Standard： 26.74 mm { 1.0528 in }

2. Direct clutch return spring sub-assembly measurement

1. Measure the direct clutch return spring sub-assembly using a vernier caliper.

Note

  • Using vernier calipers, measure the free length of the direct clutch return spring sub-assembly.
  • If the free length is smaller than the minimum, replace the direct clutch return spring sub-assembly.

Standard： 19.51 mm { 0.7681 in }

3. Reverse clutch return spring sub-assembly measurement

1. Measure the reverse clutch return spring sub-assembly using a vernier caliper.

Note

  • Using vernier calipers, measure the free length of the reverse clutch return spring sub-assembly.
  • If the free length is smaller than the minimum, replace the reverse clutch return spring sub-assembly.

Standard： 21.04 mm { 0.8283 in }

4. Clutch disc No.1 inspection

1. Inspect the clutch disc No.1.

Note

  • Check the lining of all discs
  • If the lining is flaking or discolored, or if it is worn or the print mark is wearing away, replace the discs with a new one.
  • When replacing, check the contact surfaces between the plate and the flange If they are scratched or discolored, replace the discs, plates and flanges with new ones.
  • Before replacing with new plates, soak them in ATF at least 2 hours.

7. Forward clutch assembly measurement

1. Measure the forward clutch assembly using a dial gauge.

Note

  • Measure the inside diameter of the input shaft sub-assembly bush.
  • Measure the inside diameter at different places and take an average.
  • If the measurement is greater than the maximum, replace the input shaft sub-assembly bush with a new one.
  • When the input shaft sub-assembly bush is replaced, inspect the contact surface opposed to the intermediate shaft.
  • If the surface is scratched or discolored, replace the intermediate shaft with a new one.

Standard： 20.000 to 20.025 mm { 0.7874 to 0.7884 in }
