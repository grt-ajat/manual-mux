Front case inspection (MUX)

1. Reverse idle gear inspection

1. Inspect the reverse idle gear.

Note

  • Replace parts as needed if excessive worn or damage is discovered during inspection.

2. Measure the reverse idle gear using a dial gauge.

Note

  • Measure the inner diameter of the gear using an inside dial gauge.
  • Replace the reverse idle gear if the measured value exceeds the limit.
Inner diameter of gear
Standard value： 29.980 to 30.000 mm { 1.1803 to 1.1811 in }
Limit： 30.000 mm { 1.1811 in }

2. Reverse idle gear shaft inspection

1. Inspect the reverse idle gear shaft.

Note

  • Replace parts as needed if excessive worn or damage is discovered during inspection.

2. Measure the reverse idle gear shaft using the micrometer.

Note

  • Measure the outer diameter of the reverse idle gear shaft.
  • Replace the reverse idle gear shaft if the measured value exceeds the limit.
Inner diameter of gear
Standard value： 29.926 to 29.939 mm { 1.1782 to 1.1787 in }
Limit： 29.926 mm { 1.1782 in }
