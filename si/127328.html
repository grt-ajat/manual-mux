Remote controller key setting (All models)

1. Remote controller key safety information

Caution

  • When starting the remote controller key registration operation, observe the values for "Security Wait Time" on the ECM and the ICU.
    If the values are not 0, it is necessary to maintain IGN-ON for the time of the longer "Security Wait Time" value on the ECM or the ICU.
    (For example, IGN-ON must be maintained for 10 minutes when the value on the ECM is 10 seconds and the value on the ICU is 10 minutes.)
    Also, do not perform IGN-OFF immediately after the time has passed.
  • After starting the remote controller key registration operation, do not perform IGN-OFF until commanded on the scan tool.
    (Wait Time will be reset.)
  • If the above is not followed and the operation is forced, it will cause an additional increase in the Security Wait Time.

2. Remote controller key initial registration

Note

  • Clear all of the registered ID code, and then register a new ID code.
  • The following is for when using a scan tool

Connect a scan tool to the DLC.

Turn ON the ignition switch.

Select the items on the scan tool screen in the following order.

> Diagnostics
> Body
> BCM
> Remote Keyless Entry Transmitter
> Remote Keyless Entry Transmitter Registration Delete

Clear the ID code.
Then, verify that the value displayed for "The number of registered keys" is "0".

Select "Remote Keyless Entry Transmitter Registration".

Follow the directions on the screen and verify that the "Registration Complete" message displays.

With the remote controller key inserted into the ignition switch, press the Lock button or the Unlock button 5 times.
(The remote controller key operation must be performed within the given time written in "Registration Mode Timeout".)

Verify that the value displayed for "The number of registered keys" has been added.
Pull out the remote controller key from the ignition switch and verify that the Lock/Unlock buttons operate properly.

Note

  • The following is for when not using a scan tool

･ Close all the doors.
･ UNLOCK all the door locks.
･ Pull out the remote controller key from the ignition switch.

Open the driver side front door, and insert the remote controller key into the ignition switch.

Repeatedly turn the ignition switch ON and OFF 3 times within 5 seconds.

After that, repeat closing and then opening the driver side front door 2 times within 5 seconds.

Then, repeat turning the ignition switch ON and OFF 2 times within 10 seconds.

Pull out the remote controller key from the ignition switch and verify that the Lock/Unlock buttons operate properly.

3. Remote controller key additional registration

Note

  • Register the new ID code received to the registered ID code as additional code.

For additional registrations, when the total registered and newly registered ID codes exceed 5, they will be deleted starting with the oldest code.

Note

  • The following is for when using a scan tool

Connect a scan tool to the DLC.

Turn ON the ignition switch.

Select the items on the scan tool screen in the following order.

> Diagnostics
> Body
> BCM
> Remote Keyless Entry Transmitter
> Remote Keyless Entry Transmitter Registration

Follow the directions on the screen and verify that the "Do you want to register another Transmitter?" message displays.

With the remote controller key inserted into the ignition switch, press the Lock button or the Unlock button 5 times.
(The remote controller key operation must be performed within the given time written in "Registration Mode Timeout".)

Verify that the value displayed for "The number of registered keys" has been added.
Pull out the remote controller key from the ignition switch and verify that the Lock/Unlock buttons operate properly.

Note

  • The following is for when not using a scan tool

･ Close all the doors.
･ UNLOCK all the door locks.
･ Pull out the remote controller key from the ignition switch.

Open the driver side front door, and insert the remote controller key into the ignition switch.

Repeatedly turn the ignition switch ON and OFF 3 times within 5 seconds.

After that, close and then open the driver side front door 1 time within 5 seconds.

Then, turn the ignition switch ON and OFF 1 time within 10 seconds.

If the hazard light flashes 3 times, the ID code can be recorded.

Press the Lock button or Unlock button on the remote key within approximately 12 seconds.

When recording is finished, the hazard light will flash 1 time.

Note

  • When performing multiple recordings, press the Lock button or Unlock button on the other remote keys within approximately 12 seconds.

The hazard light will flash 3 times and ID code recording will be completed under the following conditions.

・When the number of recorded ID codes is 5.
・When the ignition switch is ON.
・When 12 seconds have passed since ID code recording started.

Note

  • ID Code confirmation
  • Verify the number of registered ID codes with this mode.

･ Close all the doors.
･ Unlock all the door locks.
･ Pull out the remote controller key from the ignition switch.

Repeat inserting the remote controller key into the ignition switch and then pulling it out 2 times, and then leave the remote controller key inserted on the third time.
(This procedure must be completed within 5 seconds.)

The number of flashes of the hazard lights at this time is the number of registered ID codes.
