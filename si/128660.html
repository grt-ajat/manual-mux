Diagnostic system check-immobilizer controls (Immobilizer)

1. Diagnostic system check-immobilizer controls description of function

Diagnostic system check-immobilizer controls is a systematic method for verifying problems caused by the system malfunctions of the immobilizer system and is the starting point for all the diagnostic activities. By correctly using these diagnostic steps, it is possible to reduce the time for diagnosis and prevent replacing parts that are not defective.

2. Diagnostic system check-immobilizer controls

Note

  • If there are no complaints regarding drivability, do not perform these diagnostic steps unless otherwise instructed to do so.
  • Verify the service technical report before starting the diagnosis.
  • Do not clear the DTCs unless it is instructed in a diagnostic step.
  • The battery should be fully charged.
  • The battery cable should be normal and securely connected.
  • The ICU ground should be securely connected to the correct position.
  • The ICU harness connector should be clean and correctly connected.
  • The immobilizer security information must be correctly programmed into the ICU and ECM.
  • The ICU should be installed at the steering lock portion properly.
  • There is nothing interfering with the transponder key in the steering lock section.
  • There is no other transponder key installed to the key holder.

1. Inspection of scan tool power up

Install a scan tool.

Verify whether the scan tool powers up.

If it does not power up, refer to "Scan tool does not power up" of "Description Engine" section, and repair as necessary.

2. Scan tool communication inspection

Turn ON the ignition switch.

Verify the vehicle specifications and the status of communication with the following control units with the scan tool.

ECM
TCM (A/T TB-50LS only)
TCCM (4WD model)
EHCU (if so equipped)
ICU
SRS control unit (if so equipped)
BCM (if so equipped)
Instrument panel cluster

If communications can be established with all control modules, go to Data display confirmation.

If there are any control modules with which communication has not been established, refer to "Scan tool does not communicate with CAN device" of "Description Engine" section, and repair as necessary.

3. Data display confirmation

Verify that Ignition Status display coincides with ON and OFF of the ignition switch with a scan tool.

If the Ignition Status display is correct, go to DTC verification.

If the Ignition Status display is not correct, go to Open circuit inspection of ignition power supply circuit.

4. Open circuit inspection of ignition power supply circuit

Inspect the Meter 10 A (IG) fuse, and repair as necessary. If the fuse blows again, repair the short to ground in the circuit relating to the Meter 10 A (IG) fuse.

If the fuse is normal, repair any open circuit or high resistance in the ignition power supply circuit.

5. DTC verification

Observe the DTC information for the following control module on a scan tool. 

ECM
TCM (A/T TB-50LS only)
TCCM (4WD model)
EHCU (if so equipped)
ICU
SRS control unit (if so equipped)
BCM (if so equipped)

If a DTC beginning with U or a DTC related to communication with other control modules has been detected, diagnose the applicable DTCs.

If ICU DTC B0001, B0002, or B0012 has been detected, diagnose the applicable DTC.

If a DTC other than the above DTC has been detected, diagnose the applicable DTC.

6. Confirmation 1 of symptom

Verify the customer's concern is related with the following systems.

Engine
Transmission (A/T TB-50LS only)

If it is applicable, refer to Diagnostic system check for the relevant system and inspect it.

7. Confirmation 2 of symptom

Crank the engine.

If the engine does not crank, refer to "Engine does not turn over" in the "Description Engine" section to perform the inspection.

Start the engine.

If the engine does not start, refer to the "Starting system check" in the "Description Engine" section to perform the inspection.
