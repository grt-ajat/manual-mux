Upper control arm (2WD high ride suspension) (4WD) installation (All models)

1. Bushing installation

1. Install the bushing to the upper control arm using special tool.


SST: 5-8840-0257-0 - bush remover & installer


SST: 5-8840-2327-0 - installer; upper and lower arm bushing

  1. 5-8840-0257-0
  2. 5-8840-2327-0

2. Upper ball joint connect

1. Install the upper ball joint to the upper control arm.

Tightening torque： 31 N・m { 3.2 kgf・m / 23 lb・ft }

2. Connect the upper ball joint to the knuckle.

Tightening torque： 98 N・m { 10.0 kgf・m / 72 lb・ft }

Note

  • Install so that it fits in the cotter pin hole.
  • Install the nut and cotter pin, and then tighten at the specified torque.
  • Use a new cotter pin.

3. Upper control arm installation

1. Install the upper control arm to the frame.

Caution

  • Apply ISO: VG46 oil or equivalent to the threaded portions of the bolt and the nut.

Tightening torque： 137 N・m { 14.0 kgf・m / 101 lb・ft }

Note

  • Tighten the bolt together with the part at the location described in the illustration below.
  • Measure the buffer clearance.

Standard： 26.0 mm { 1.02 in }

  1. Lower control arm
  2. Buffer clearance
  3. Bumper rubber
  4. Center of rear side cam bolt
  5. Trim height

2. Install the brake hose to the upper control arm.

4. Front wheel speed sensor installation

1. Install the wheel speed sensor to the knuckle.

Caution

  • When installation, be careful for the wheel speed sensor not to hit against anything.

Note

  • Tighten the sensor at the specified torque.

Tightening torque： 8 N・m { 0.8 kgf・m / 71 lb・in }

2. Connect the harness clip to the harness.

Note

  • Install the harness fixing bolt, and tighten it to the specified torque.

Tightening torque： 8 N・m { 0.8 kgf・m / 71 lb・in }

Note

  • Install the harness fixing nut, and tighten it to the specified torque.

Tightening torque： 20 N・m { 2.0 kgf・m / 15 lb・ft }

Note

  • Connect the harness clip indicated by the arrow.

  1. Connector position
  2. Wheel speed sensor assembly
  3. Harness tightening nut
  4. Harness tightening bolt
  5. Sensor tightening bolt

Caution

  • When installing the harness, check the harness for a twist.

5. Disc wheel installation

1. Install the disc wheel to vehicle.

Note

  • Install the wheel nuts in the numbered order.

Tightening torque: 120 N⋅m {12.0 kgf⋅m/87 lb⋅ft}
