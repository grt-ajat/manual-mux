ECM setting (4JJ1)

1. ECM safety information

Caution

  • When replacing the ECM, the vehicle data must be written into the new ECM.
  • When replacing the ECM, after removal download the old ECM data to the new ECM, referring to the following.
  • Use the same scan tool for the replacement procedure from beginning until end. If the scan tool is changed during the procedure, the correct information cannot be written.
  • If the scan tool is changed during the procedure, the correct information cannot be written.

2. ECM writing

ECM writing (PESS control unit)

Note

  • If equipped with a PESS, this procedure is required.

1. Connect a scan tool to the DLC.

2. Turn ON the ignition switch.

3. Select the Diagnostics.

4. Select the Body.

5. Select the PESS.

6. Select the Programming.

7. Select Program PESS Function.

Note

  • Write the immobilizer security information into the BCM by following the on-screen instructions.

Caution

  • If an incorrect security code is input, the security wait time begins.
  • When turning the ignition switch to ON or OFF, wait at least 5 seconds.
  • Keep the transponder keys to be programmed away from other keys, other transponder keys, key number plates, or other metal objects.

8. Turn OFF the ignition switch.

Note

  • After the programming is completed, turn the ignition switch OFF for 1 minute.

9. Turn ON the ignition switch.

Note

  • Verify that the engine starts with all transponder keys.

ECM writing (ICU)

Note

  • If an immobilizer is equipped, this procedure is required.

10. Connect a scan tool to the DLC.

11. Turn ON the ignition switch.

12. Select the Diagnostics.

13. Select the Body.

14. Select Immobilizer.

15. Select Programming.

16. Select Program Immobilizer Function.

Note

  • Follow the directions on the screen and write the immobilizer security information into the ECM.

Caution

  • If an incorrect security code is input, the security wait time begins.
  • When turning the ignition switch to ON or OFF, wait at least 5 seconds.
  • Keep transponder keys to be programmed away from other keys, other transponder keys, key number plates, and other metal parts.

17. Turn OFF the ignition switch.

Note

  • After the programming is completed, turn OFF the ignition switch for 1 minute.

18. Turn ON the ignition switch.

Note

  • Verify that the engine does not start with all transponder keys.

3. ECM setting

1. Connect a scan tool to the DLC.

2. Select the Diagnostics.

3. Select the Engine.

4. Select the engine model.

5. Select Programming.

6. Select the Program VIN (Vehicle Identification Number).

7. Select the Program VIN (Vehicle Identification Number).

Note

  • If the VIN does not display or is not correct, input the correct VIN.

8. Select the Program VIN (Vehicle Identification Number).

Note

  • Follow the directions on the screen and write the VIN into the ECM.
  • When the ECM to be replaced can communicate with a scan tool, do not perform the following procedure.

9. Select the Injector ID Code.

10. Select the Injector Flow.

Note

  • Follow the directions on the screen and write the injector ID codes into the ECM.

11. Select Vehicle Speed Correction Data.

Note

  • Write the Vehicle Speed Correction Data to the ECM by following the instructions on the screen.

12. Turn OFF the ignition switch.

Note

  • After the programming is completed, turn OFF the ignition switch for 1 minute.

4. ECM download

Note

  • When the ECM to be replaced can communicate with a scan tool, download the injector ID codes.

Caution

  • Confirm the vehicle specifications before downloading.
  • Perform the download after installing the new ECM in the vehicle.

1. Connect a scan tool to the DLC.

2. Turn ON the ignition switch.

3. Select the Diagnostics.

4. Select the Engine.

5. Select the engine model.

6. Select Programming.

7. Select the Injector ID Code.

8. Select the Injector ID Download.

Note

  • Follow the directions on the screen and download the injector ID codes to the ECM.

9. Select the Injector ID Code.

10. Select the Injector Flow.

Caution

  • Confirm that the injector ID codes displayed on the screen match the injector ID codes recorded before removing the ECM.

Note

  • If the injector ID codes do not display or are not correct, input the correct injector ID codes.

11. Turn OFF the ignition switch.

Note

  • After the downloading is completed, turn OFF the ignition switch for 1 minute.
