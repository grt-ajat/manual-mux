DTC B1001 Option Configuration Error (BCM with PESS)

1. DTC B1001 description of DTC

The BCM compares the vehicle information in the application area to the programmed vehicle information when power up. If these programmed vehicle information do not match, this DTC is set.

2. DTC B1001 condition for setting the DTC

The Current Value(s) of Programming & Resetting is not registered.
