Instrument panel cluster setting (All models)

1. Instrument panel cluster safety information

Caution

  • When replacing the instrument panel cluster, the vehicle information must be written into the new instrument panel cluster.
  • When replacing the instrument panel cluster, before removal upload the old instrument panel cluster data to a scan tool, referring to the following.
  • When replacing the instrument panel cluster, download the data from the old instrument panel cluster to the new instrument panel cluster, referring to the following.

2. Instrument panel cluster download

Note

  • If communication between the old instrument panel cluster and a scan tool is possible before replacement, download the vehicle data.

Caution

  • Make sure to check the vehicle specifications before performing the download.
  • Perform the download after installing the new instrument panel cluster in the vehicle.

1. Turn ON the ignition switch.

2. Turn ON a scan tool.

3. Select the Diagnostics.

4. Select the body.

5. Select the meter.

6. Select the programming.

7. Select Download Vehicle Configuration Data.

Note

  • Follow the directions on the screen to download the vehicle data to the instrument panel cluster.

8. Select Module Information.

Caution

  • Refer to the following list to confirm the download setting value.

3. Instrument panel cluster setting

Note

  • Clock Display Setting

1. Turn ON the ignition switch.

2. Turn ON a scan tool.

3. Select the Diagnostics.

4. Select the Body.

5. Select the meter.

6. Select the Programming.

7. Select Clock Display Setting

Note

  • Follow the directions on the screen to program the data in the instrument panel cluster.
  • The clock will be displayed within the instrument panel cluster when Available is selected.
  • Select Available in models without an Isuzu genuine audio system.

8. Turn OFF the ignition switch.

Note

  • Turn OFF the ignition switch for 30 seconds.

9. Turn ON the ignition switch.

10. Check display of the meter.

Note

  • Vehicle Configuration Data Setting
  • When communication between the old instrument panel cluster and a scan tool is not possible to be carried out, program the confirmed vehicle data above individually.

11. Turn ON the ignition switch.

12. Turn ON a scan tool.

13. Select the Diagnostics.

14. Select the body.

15. Select the meter.

16. Select the programming.

17. Select Vehicle Configuration Data Setting.

Note

  • Follow the directions on the screen to program the data in the instrument panel cluster.

Caution
