High mounted stop light installation (All models)

1. High mounted stop light installation

1. Install the gasket to the high mounted stop light.

2. Connect the connector to the high mounted stop light.

  1. Connector
  2. High mounted stoplight

3. Install the high mounted stop light to vehicle.

Note

  • Insert the high mounted stoplight until the rib portion on the top of the stoplight engages with the tail gate panel as shown in the diagram.
  • Engage the 3 high mounted stoplight springs with the tail gate panel to install the high mounted stoplight.

4. Install the cover to the tail gate.

  1. Cover
  2. Spring

2. Battery ground cable connect

1. Connect the battery ground cable to the battery.

Note

  • Check that the light illuminates.
