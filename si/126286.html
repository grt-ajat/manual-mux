DTC P0016 (Flash Code 16) Crankshaft Position - Camshaft Position Correlation (4JJ1)

1. DTC P0016 diagnostics

Note

  • This DTC is set by an incorrect mechanical timing condition, which is most likely caused by incorrect installation of the timing gear or chain.
  • Engine reverse rotation may set this DTC.

1. Diagnostic system check

Refer to "202.Description Engine 15C.Functional Inspection(4JJ1) diagnostic system check-engine controls".

2. Prioritized DTC

If DTCs P0335, P0336, or P0340 are set at the same time, go to the applicable DTC diagnosis.

3. Inspection of sensor

Inspect the CKP sensor and the CMP sensor for the following conditions, and repair as necessary. Go to Repair verification.

Sensor damage.
Loose or incorrectly installed sensor.
Excessive gap.
Foreign material passing between sensor and sensor rotor or chain sprocket.
Sensor rotor or chain sprocket damage.
Loose or incorrectly installed sensor rotor or chain sprocket.

If it is normal, go to Inspection of engine mechanical timing.

4. Inspection of engine mechanical timing

Inspect the engine mechanical timing for the following conditions, and repair as necessary. Go to Repair verification.

Incorrectly installed timing gear or chain.
Timing chain tensioner malfunction.
Excessive play in timing chain.
Timing chain that has jumped teeth.

2. DTC P0016 repair verification

Clear the DTC with a scan tool.

Turn OFF the ignition switch for at least 30 seconds.

Start the engine.

Note

  • If the engine does not start, crank the engine for 10 seconds.

Operate the vehicle within the conditions for setting the DTC or within the parameters observed in the Freeze Frame.

Observe the DTC information with a scan tool.
