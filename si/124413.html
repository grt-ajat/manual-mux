Diagnostic system check-passive entry and start system (Passive Entry and Start System)

1. Diagnostic system check-passive entry and start system description of function

Diagnostic system check-Passive Entry and Start System (PESS) controls is a systematic method for verifying problems caused by the system malfunctions of the PESS and is the starting point for all the diagnostic activities. By correctly using these diagnostic steps, it is possible to reduce the time for diagnosis and prevent replacing parts that are not defective.

2. Diagnostic system check-passive entry and start system

Note

  • If there are no complaints on drivability, do not perform these diagnostic steps unless otherwise instructed in other sections.
  • Check the service technical report before starting the diagnosis.
  • Do not clear the DTCs unless it is instructed in a diagnostic step.
  • The battery should be fully charged.
  • The battery cable should be normal and securely connected.
  • The ground of the PESS control unit should be securely connected to the correct position.
  • The PESS control unit harness connector should be clean and correctly connected.
  • The PESS control unit terminals should be clean and matched correctly.
  • Immobilizer security information must be correctly programmed into the PESS control unit and ECM.

1. Inspection of scan tool power up

Install a scan tool.

Verify whether the scan tool powers up.

If it does not power up, refer to "Scan tool does not power up" in the Description Engine section, and repair as necessary.

2. Inspection 1 of scan tool communication

Perform one of the following operations.

Press the engine start/stop button.
Grip the driver side outside handle.
Press the driver side outside handle switch.
Press the tail gate opener switch.
Press the tail gate lock button.
Open the door and close the door.

Verify that communication can be established with the PESS control units using a scan tool.

If communication can be established, go to Inspection of power mode switching.

If communication cannot be established, go to Lost communication with the PESS control unit.

3. Inspection of power mode switching

Verify that the power mode can be switched to the ACC or the ON state.

If switching to the ACC or the ON state is possible, go to Inspection 2 of communication with scan tool.

If switching to the ACC or the ON state is not possible, refer to "The power mode does not change" in the Description Security and Locks Control_Symptom, and repair as necessary.

4. Inspection 2 of scan tool communication

Turn ON the ignition switch.

Verify the vehicle specifications and the status of communication with the following control units with the scan tool.

ECM
TCM (A/T TB-50LS only)
TCCM (4WD models)
EHCU
SRS control unit
BCM
Instrument panel cluster

If communication can be established with all control modules, go to DTC verification.

If there are any control modules with which communication cannot be established, refer to Scan tool does not communicate with CAN device in the Description Engine section, and repair as necessary.

5. DTC verification

Observe the DTC information for the following control modules with a scan tool.

ECM
BCM
PESS control unit

If a DTC beginning with U or a DTC related to communication with other control modules is set, diagnose the applicable DTC.

If the PESS control unit DTC B1007, B1009 or B1014 is set, diagnose the applicable DTC.

If PESS control unit DTC B1511 has been detected, diagnose the applicable DTC.

If a DTC other than the above DTC has been detected, diagnose the applicable DTC.

6. Confirmation 1 of symptom

Verify the customer's concern is related with the following systems.

Engine
Body control

If it is applicable, refer to Diagnostic system check for the relevant system and inspect it.

7. Confirmation 2 of symptom

Crank the engine.

If the engine does not crank, refer to "Engine does not turn over" in the "Description Engine" section to perform the inspection.

Start the engine.

If the engine does not start, refer to the "Starting system check" in the "Description Engine" section to perform the inspection.

8. Verification 3 of symptom

Operate the vehicle and check if the following conditions are met.

Doors do not unlock even when gripping the driver side door handle
Doors do not lock even when operating the door lock button on the driver side
Doors do not lock or unlock with electronic key button operation
Doors do not unlock even when opener switch is operated
Doors do not lock even when tail gate lock button is operated
Answer back buzzer (external buzzer) does not work
No electronic key warning buzzer (external buzzer) does not work

If a problem is found, repair the applicable symptoms.

If none of the symptoms apply, refer to Intermittent conditions of passive entry & start system, and inspect as necessary.
