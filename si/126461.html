Hard start (4JJ1)

1. Hard start description of symptom

The engine cranks but does not start for a long time. The engine does eventually run, or may start but immediately stops.

2. Hard start diagnostics

Note

  • Preliminary inspection

Inspect the engine control system.
Verify that the driver is using the correct starting procedure.
Ensure that the ground of the ECM is free of dirt and is fixed securely in the correct position.
Inspect whether the harness connector is correctly connected.
Inspect the fuel type, quality, and amount.
Verify the injector ID code of each cylinder.
Check the scan tool data list.
Check for related Service Bulletins.

Note

  • Sensor inspection

While the engine is cold, compare each display for Engine Coolant Temperature (ECT), Intake Air Temperature (IAT), and Fuel Temperature to the scan tool data list.
If the difference between the temperature values is 5 °C (9 °F) or more when the engine has cooled, inspect for high resistance in each circuit and for sensor failure.
{When the ignition switch is turned ON, the MAF sensor is heated, and as a result, the IAT shows a higher value than the normal intake air temperature.}
Observe the Fuel Rail Pressure (FRP) Sensor display on the scan tool. It is normal if the Fuel Rail Pressure (FRP) Sensor display shows 0.9 - 1.0 V when the engine is not started with the ignition switch turned ON after at least 1 minute from the stop of engine operation. If not, inspect for high resistance in each circuit and for any sensor failure.
The CKP sensor is securely installed, and the sensor rotor is not damaged.
The CMP sensor is securely installed, and the camshaft gear is not damaged.

Note

  • Fuel system inspection

Inspect for air in the fuel system.
Inspect for water in the fuel.
Inspect for external fuel leakage and excessive engine oil.
Inspect if the fuel is waxy or freezing.
Inspect whether the fuel filter indicator light turns on continuously or frequently when the engine is running.
Inspect the fuel filter element for clogging.
Inspect for fuel leakage from the pressure limiter and the injector.
Inspect the fuel system for collapsing, twisting, improper tightening, cracking, or clogging.
Inspect the vent valve and the hose of the fuel tank for clogging.
{The piping may be clogged due to foreign material from the fuel tank having entered the fuel system.}
Inspect the fuel supply pump operation.
{The fuel supply pump must be timed with the engine, and the adjusted value should be written into the ECM.}
Perform the Cylinder Balance Test with a scan tool.
Perform the Injector Forced Drive with a scan tool.
Observe the Fuel Rail Pressure (FRP) Regulator Valve Feedback Current display on the scan tool.

Note

  • Air intake system inspection

Inspect the air cleaner and air intake duct for clogging, holes and leakage.
Inspect for any condition that restricts the flow within the turbocharger inlet duct.
Inspect for stuck-close intake throttle valve.
Inspect for clogging and leakage in the intake manifold.

Note

  • Exhaust system inspection

Inspect for any condition that restricts the flow of the exhaust system.

Note

  • Engine mechanical inspection

Inspect the compression pressure of the engine.
Check for improper idle gear timing.
Inspect the valve clearance.
Inspect the valve spring for damage and fatigue.
Inspect the camshaft for worn.
Inspect the ring gear for worn.
Inspect the inside of the engine for seizure.

Note

  • Electrical system inspection

Inspect the glow plug control system operation.
Inspect the cranking speed.
Inspect the battery for deterioration.
