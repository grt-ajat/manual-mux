Rear disc brake inspection (All models)

1. Rear disc brake inspection

1. Inspect the brake assembly.

Note

  • If wear, damage, or other abnormal conditions are found during inspection, repair or replace the defective parts.
  • Visually inspect the following parts for wear, bending, deterioration, corrosion, scratching, and other abnormal conditions.
  • Rotor
  • Caliper body
  • Cylinder bore
  • Piston
  • Brake support
  • Lock bolt
  • Guide pin

2. Rear rotor inspection

1. Inspect the brake rotor.

Note

  • Measure the thickness of the brake rotor.

Standard： 18.0 mm { 0.709 in }

Limit： 16.6 mm { 0.654 in }

Note

  • Measure the runout of the brake rotor.

Standard： 0.075 mm { 0.00295 in }

Note

  • Correctly adjust the wheel bearing before inspection.
  • Using a dial gauge, measure runout at a location 10 mm {0.04 in} from the outer edge of the rotor surface that makes contact with the disc brake pad.

Note

  • Measure the degree of parallelization of the brake rotor.

Limit： 0.023 mm { 0.00091 in }

Note

  • The contact surface is within 0.023 mm {0.0009 in} of the position located 10 mm {0.04 in} inward from the rotor outer end.
  • Measure at 8 or more locations in the circumferential direction.

Note

  • Measure the inner diameter of the parking brake drum.

Standard： 210.0 mm { 8.268 in }

Limit： 211.5 mm { 8.327 in }

Note

  • Measure the drum runout.

Standard： 0.08 mm { 0.0031 in }

Limit： 0.15 mm { 0.0059 in }

3. Disc brake pad inspection

1. Inspect the disc brake pad.

Note

  • Measure the thickness of the disc brake pad.

Standard： 10.0 mm { 0.394 in } Rear disc brake

Limit： 1.5 mm { 0.059 in } Rear disc brake

Note

  • When the pad wear indicator makes a scratching sound or is worn to within 1.8 mm {0.071 in} of the shoe table, replace the pad.

Caution

  • When replacing, replace all 4 pads (inner and outer pads of both the left and right wheels).

4. Brake caliper assembly inspection

1. Inspect the guide pin.

2. Inspect the lock pin.

Note

  • Inspect for sticking, wear, bending, deterioration, corrosion, scratches, or other abnormal conditions.
  • Repair or replace if abnormal conditions are found.

  1. Support bracket
  2. Slide pin boot
  3. Slide pin (Guide pin)
  4. Lock bolt
  5. Pad clip
  6. Slide pin (Lock pin)

3. Inspect the boot.

Note

  • Replace the boot every time the caliper is overhauled.

4. Inspect the dust seal.

Note

  • Replace the dust seal and piston seal every time the caliper is overhauled.
