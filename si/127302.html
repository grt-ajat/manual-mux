Rear door assembly reassembly (All models)

1. Door harness installation

1. Install the door harness to the rear door.

2. Outside handle installation

1. Install the outside handle to the rear door.

Note

  • Tighten the fixing bolts to the specified torque.

Tightening torque： 9 N・m { 0.9 kgf・m / 80 lb・in }

3. Door lock assembly installation

1. Install the door lock assembly to the rear door.

Note

  • Install the 3 fixing screw and the bolt.
  • Connect the linkage and harness connector.
  • Tighten the fixing screws and bolt to the specified torque.

Tightening torque： 7 N・m { 0.7 kgf・m / 62 lb・in }

4. Glass run installation

1. Install the glass-run to the rear door.

5. Glassrun rear channel lower installation

1. Install the glassrun rear channel lower to the rear door.

Note

  • Install the 2 fixing bolts.

6. Rear door window regulator installation

1. Install the window regulator to the rear door.

Note

  • Install the 4 setting bolts.
  • Tighten the fixing bolts to the specified torque.

Tightening torque： 9 N・m { 0.9 kgf・m / 80 lb・in }

Note

  • Connect the power window motor connector.

7. Rear door glass installation

1. Install the rear door glass to the rear door.

Note

  • Install the rear door glass to the window regulator with the 2 bolts.
  • Tighten the fixing bolts to the specified torque.

Tightening torque： 9 N・m { 0.9 kgf・m / 80 lb・in }

8. Fix window assembly installation

1. Install the fix window assembly to the rear door.

9. Glassrun rear channel upper installation

1. Install the glassrun rear channel upper to the rear door.

10. Outer waist seal installation

1. Install the outer waist seal to the rear door.

11. Inside lever installation

1. Connect the inside lever to the door lock cable.

12. Weather strip installation

1. Install the weather strip to the rear door.

Note

  • Install the clip.

13. Waterproof sheet installation

1. Install the waterproof sheet to the rear door.

14. Door speaker installation

1. Install the door speaker to the rear door.

Note

  • Connect the connector and install the fastening bolt.

15. Bracket installation

1. Install the bracket to the rear door.

Note

  • Install the 2 screws.

16. Rear door trim panel installation

1. Install the rear door trim panel to the rear door.

Note

  • Insert the 9 clips for the rear door trim panel, and tighten the screw.
  • Insert the 2 clips for the cover.
  • Tighten the fixing screws to the specified torque.

Tightening torque： 2 N・m { 0.2 kgf・m / 18 lb・in }

  1. Screw
  2. Clip
  3. Cover

17. Power window switch installation

1. Install the power window switch to the rear door.

Note

  • Connect the connector, and install the power window switch.
  • Install the screws and close the cover.

  1. Power window switch
  2. Screw
  3. Cover
