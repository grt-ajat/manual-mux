Diagnostic system check- ABS/EBD/TCS/ESC controls (All models)

1. Diagnostic system check- ABS/EBD/TCS/ESC controls description of function

The Diagnostic system check - ABS/EBD/TCS/ESC controls is an organized approach to identifying a problem created by an ABS system malfunction, and must be the starting point for any drivability complaint diagnosis. Using it correctly will reduce diagnostic time and prevent the replacement of good parts.

2. Diagnostic system check- ABS/EBD/TCS/ESC controls

Note

  • If there are no complaints regarding drivability, do not perform these diagnostic steps unless otherwise instructed to do so.
  • Check the service technical report before starting the diagnosis.
  • Do not clear the DTCs unless it is instructed in a diagnostic step.
  • The battery should be fully charged.
  • The battery cable should be normal and securely connected.
  • The EHCU ground should be securely connected in the correct position.
  • The EHCU harness connector should be clean and correctly connected.
  • The EHCU terminal should be clean and matched correctly.

1. Inspection of scan tool power up

Install a scan tool.

Verify whether the scan tool powers up.

If the scan tool does not power up, refer to Scan tool does not power up in the Description Engine section, and repair as necessary.

2. Inspection of scan tool communication

Install a scan tool.

Turn ON the ignition switch.

Verify the vehicle specifications and the status of communication with the following control units with the scan tool.

ECM
TCM (A/T TB-50LS only)
TCCM (4WD model)
EHCU
ICU (if so equipped)
SRS control unit (if so equipped)
BCM (if so equipped)
Instrument panel cluster

If communication can be established with all control modules, go to Inspection of ABS and ESC warning lights.

If there are any control modules with which communication cannot be established, refer to Scan tool does not communicate with CAN device in the Description Engine section, and repair as necessary.

3. Inspection of ABS and ESC warning lights

Turn OFF the ignition switch.

Turn ON the ignition switch.

Verify the ABS and ESC warning lights.

Note

  • The ABS and ESC warning lights illuminate for approx. 3 seconds after the ignition switch is turned ON, and then goes OFF.

If the ABS and ESC warning lights do not illuminate, refer to Check ABS, ESC, brake system warning light inoperative, and repair as necessary.

If the ABS and ESC warning lights do not turn OFF, refer to Check ABS, ESC, brake system warning light always on, and repair as necessary.

4. DTC verification

Observe the DTC information for the following control modules with a scan tool.

EHCU
ECM
TCM (A/T TB-50LS only)
TCCM (4WD model)

If DTCs beginning with U or DTCs related to communication with other control modules are set, diagnose the applicable DTCs.

If EHCU DTCs C0550 or C0560 are set, diagnose the applicable DTCs.

If EHCU DTC C0800 is set, diagnose the applicable DTC.

If DTCs other than the above DTCs are set, diagnose the applicable DTC.

5. Confirmation 1 of symptom

Verify the customer's concern is related with the following systems.

Engine
Transmission (A/T TB-50LS only)
Transfer

If applicable, refer to Diagnostic system check for the applicable system, and inspect as necessary.

6. Confirmation 2 of symptom

Start the engine.

Verify that the ESC OFF switch can be operated.

Press the ESC OFF switch for at least 5 seconds and if the TCS OFF indicator light and ECS OFF indicator light do not illuminate, refer to ESC OFF switch circuit system check, and repair as necessary.

Drive the vehicle and check if the following conditions are met.

Wheels lock when braking
Uneven braking occurs while ABS works
Brake pedal feeling is abnormal
Brake operation sounds are heard from the hydraulic unit even though the brake is not applied
One wheel locks
Dragging brakes
Brake abnormal noise
Bake does not work efficiently
Brake pedal pulsates
Stepping power of the brake pedal is large
Stroke of the brake pedal is abnormal
Defective straight driving
ABS works frequently but the brake is not effective or TCS operates frequently
ESC does not work efficiently

If a problem is found, repair the applicable symptoms.

If any of the symptoms are not met, refer to Intermittent conditions of brake, and inspect.
