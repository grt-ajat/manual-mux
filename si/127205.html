Center ventilation grille installation (All models)

1. Center ventilation grille installation

1. Install the center ventilation grille to the instrument panel.

Note

  • Put the center ventilation grille in, and install the screws.

2. Radio installation

1. Install the radio side bracket to the radio.

Tightening torque： 2.0 N・m { 0.2 kgf・m / 18 lb・in }

Note

  • Securely insert the claw section of the bracket into the radio, and fix it with the screws.

2. Connect the harness connector to the radio.

3. Install the radio to the instrument panel.

Tightening torque： 2.0 N・m { 0.2 kgf・m / 18 lb・in }

  1. Clip
  2. Radio
  3. Screw

3. Audio and navigation system installation

Note

  • Audio and navigation system equipped model
  • Install the USB to the shift console cover.
  • Install the USB cable to the USB.

  1. USB
  2. Shift console cover
  3. USB cable

1. Install the bracket to the audio and navigation system.

Tightening torque： 2.0 N・m { 0.2 kgf・m / 18 lb・in }

Note

  • Securely insert the claw section of the bracket into the audio and navigation system, and fasten the screws.

2. Connect the harness connector to the audio and navigation system.

3. Install the audio and navigation system to the instrument panel.

Note

  • Install the 4 corner clips and push them in.
  • Install the 2 screws.

Tightening torque： 2 N・m { 0.2 kgf・m / 18 lb・in }

  1. Audio and navigation system
  2. Clip
  3. Screw

4. Control panel assembly installation

Note

  • The following applies to auto A/C

1. Install the control panel assembly to the instrument panel.

Note

  • Connect 1 harness connector.
  • Install the clips at 4 corners and push them.

  1. Instrument panel
  2. Control panel assembly
