Front tongue side seat belt installation (All models)

1. Front tongue side seat belt installation

Note

  • Models without airbag

1. Install the front tongue side seat belt to the center pillar.

Note

  • Install the 2 bolts.

Tightening torque： 8.2 N・m { 0.8 kgf・m / 73 lb・in } Upper bolt

Tightening torque： 46 N・m { 4.7 kgf・m / 34 lb・ft } Lower bolt

  1. Lower bolt
  2. Upper bolt
  3. Front tongue side seat belt

Note

  • The following applies to models with pretensioners.

2. Install the front tongue side seat belt to the center pillar.

Note

  • Connect the harness connector.
  • Install the 2 bolts.

Tightening torque： 8.2 N・m { 0.8 kgf・m / 73 lb・in } Upper bolt

Tightening torque： 46 N・m { 4.7 kgf・m / 34 lb・ft } Lower bolt

  1. Lower bolt
  2. Upper bolt
  3. Front tongue side seat belt

Note

  • The following applies to models with lap pretensioners.

3. Install the front tongue side seat belt to the center pillar.

Note

  • Connect the harness connector.
  • Install the 2 bolts.

Tightening torque： 8.2 N・m { 0.8 kgf・m / 73 lb・in } Upper bolt

Tightening torque： 46.0 N・m { 4.7 kgf・m / 34 lb・ft } Lower bolt

  1. Upper bolt
  2. Front tongue side seat belt
  3. Lower bolt

Note

  • The following applies to all models.

4. Install the seat belt upper anchor bolt to the center pillar.

Tightening torque： 46 N・m { 4.7 kgf・m / 34 lb・ft }

5. Install the adjuster cover to the center pillar.

  1. Adjuster cover
  2. Seat belt upper anchor bolt

2. Center pillar lower trim cover installation

Note

  • The following applies to models with pretensioners and models without airbags.

1. Install the center pillar lower trim cover to the center pillar.

  1. Clips

Note

  • Install the seat belt lower anchor bolt to the center pillar.

Tightening torque： 46 N・m { 4.7 kgf・m / 34 lb・ft }

  1. Seat belt lower anchor bolt

Note

  • The following applies to models with lap pretensioners.

2. Install the lap pretensioner bracket to the center pillar.

Tightening torque： 46 N・m { 4.7 kgf・m / 34 lb・ft }

Note

  • Assemble bolts a and b in that order as shown in the following diagram.

  1. Lap pretensioner bracket

3. Install the lap pretensioner to the lap pretensioner bracket.

Tightening torque： 46 N・m { 4.7 kgf・m / 34 lb・ft }

Note

  • Connect the harness connector.

  1. Lap pretensioner

4. Install the center pillar lower trim cover to the center pillar.

Note

  • Move the center pillar lower trim cover and install to the bezel rubber through the lap pretensioner.

  1. Bezel rubber

Note

  • Install the center pillar lower trim cover.

  1. Clips

5. Install the tongue side seat belt to the lap pretensioner.

Note

  • Install the tongue plate to the lap pretensioner.
  • Close the cover and tighten the screw.

Tightening torque： 6 N・m { 0.6 kgf・m / 53 lb・in }

  2. Tongue plate
  3. Screw
