Introduction to the trouble diagnosis of security and locks control (Passive Entry and Start System)

1. Introduction to the trouble diagnosis of security and locks control

Non-OEM parts
All systems are adjusted to operate using genuine parts. Accordingly, if general aftermarket sensors or switches, etc., are installed, DTCs may be set due to misdiagnoses.
Before starting failure diagnoses, turn OFF or remove all aftermarket parts.

Basic knowledge
Basic system knowledge is necessary in order to effectively utilize the service manual. If performing diagnosis, a lack of basic knowledge may result in an incorrect diagnosis and damage may be caused to components. Diagnosis must not be performed without basic knowledge.

Displaying the DTC
Display the DTC with a scan tool.

Clearing the DTC
When conditions for setting the detected DTC are no longer met, the current failure DTC will be cleared.
A history DTC clears after 50 consecutive ignition cycles pass with no failures.
Clear the DTC with a scan tool.

System diagnosis
Using the diagnostic system check-Passive Entry & Start System (PESS), system diagnosis is started. The diagnostic system check-Passive Entry & Start System provides the following information:
Identification of the control modules that command the system
The ability of the control modules to communicate through the serial data circuit
Identification and conditions of detected DTCs
Using the diagnostic system check-Passive Entry & Start System (PESS), the appropriate procedure for diagnosing the system, as well as the location of that procedure are made clear.
