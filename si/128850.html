Power steering unit reassembly (All models)

1. Power steering unit reassembly

Note

  • When it is removed

1. Install the dust cover to the housing assembly.

2. Install the mounting rubber to the housing assembly.

3. Install the power steering pipe to the housing assembly.

Tightening torque： 20 N・m { 2.0 kgf・m / 15 lb・ft } Valve housing side

Tightening torque： 24 N・m { 2.4 kgf・m / 18 lb・ft } Cylinder side

4. Install the tie rod assembly to the tab washer.

Tightening torque： 108 N・m { 11.0 kgf・m / 80 lb・ft }

Note

  • After tightening, bend the tab washer against the whole width of the flat of the inner ball joint.
  • Apply grease to the tie rod groove.

5. Install the boot to the housing assembly.

Caution

  • The boot should be free from any swell or dent.
  • After installation, the narrow tip of the boot should be inserted in the tie rod groove.

6. Install the boot band to the boot using special tool.


SST: 5-8840-2745-0 - clamp plier

Gap： 2.6 to 3.3 mm { 0.10 to 0.13 in }

Caution

  • Use new boot bands.

  1. Gap of crimp

7. Install the clip to the boot.

8. Install the tie rod end to the tie rod assembly.

Tightening torque： 98 N・m { 10 kgf・m / 72 lb・ft }
