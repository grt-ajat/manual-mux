Roof rail installation (All models)

1. Front extension installation

1. Install the front extension to the body.

Note

  • Clean front extension installation area and its adhesive area at body.

Caution

  • If installation time is over than 35 minutes, those areas must be cleaned again.

Note

  • Remove the pick seal and release the paper of double adhesive tape under the front extension.

Caution

  • Color of front extension's pick seal is yellow for left side and white for right side.

  1. Pick seal
  2. Front extension

Note

  • Position the front extension in channel between the roof panel and the body side panel by aligning with the front edge of the roof panel and then apply force along length to attach it with surface.

Caution

  • In case of re-installation, the front extension has to be changed.

  1. Roof panel
  2. Front extension
  3. Body side panel
  4. 2.3 mm { 0.09 in } between front extension and windshield molding
  5. Front edge (Roof panel)
  6. Windshield molding

2. Roof rail support installation

1. Install the roof rail support to the body.

Note

  • Place the front support so that its arrow of identification mark points to the front of the vehicle. After the hole on lower surface is aligned with the 3rd body's hole from the front of the vehicle, tighten front support with the 1 bolt (M8) at that hole.

Caution

  • In case of re-installation, seal tape, sponge and bolt have to be changed.

Tightening torque： 26.5 N・m { 2.7 kgf・m / 20 lb・ft }

  1. Seal tape (Front support)
  2. Bolt (M8)
  3. Front support
  4. Sponge (Front support)

Note

  • Place the rear support so that its arrow of identification mark points to the front of the vehicle. After the hole on lower surface is aligned with the 3rd body's hole from the rear of the vehicle, tighten the rear support with the 1 bolt (M8) at that hole.

Caution

  • In case of re-installation, seal tape, sponge and bolt have to be changed.

Tightening torque： 26.5 N・m { 2.7 kgf・m / 20 lb・ft }

  1. Seal tape (Rear support)
  2. Bolt (M8)
  3. Rear support
  4. Sponge (Rear support)

3. Roof rail installation

1. Install the roof rail to the body.

Note

  • Set roof rail to the body and align with the hole on the body, the front support and the rear support. After that, assemble the roof rail by tightening the 4 bolts (M8) as sequence in illustration.

Caution

  • In case of re-installation, seal tapes, sponges and bolts have to be changed.

Tightening torque： 26.5 N・m { 2.7 kgf・m / 20 lb・ft }

  1. 1st tightened bolt (M8)
  2. 2nd tightened bolt (M8)
  3. 3rd tightened bolt (M8)
  4. 4th tightened bolt (M8)

  1. Seal tape (Roof rail front)
  2. Sponge (Roof rail front)

  1. Seal tape (Roof rail rear)
  2. Sponge (Roof rail rear)

Note

  • Tighten the 1 bolt (M6) at the front support and the 1 bolt (M6) at the rear support in the hole for the rail cover.

Caution

  • In case of re-installation, bolts have to be changed.

Tightening torque： 6.5 N・m { 0.7 kgf・m / 58 lb・in }

  1. Bolt (M6)
  2. Hole (Rail cover)

  1. Bolt (M6)
  2. Hole (Rail cover)

Note

  • Put the guide of the front cover into the roof rail and then plug the clips to lock with the roof rail.

  1. Front cover
  2. Guide (Front cover)
  3. Clip (Front cover)
  4. Roof rail

Note

  • Put the guide of the rear cover into the roof rail and then plug the clips to lock with the roof rail.

  1. Guide (Rear cover)
  2. Clip (Rear cover)
  3. Rear cover
  4. Roof rail

4. Rail cover installation

1. Install the rail cover to the roof rail.

Note

  • Plug the rail cover into the holes on the roof rail which are above the front support and above the rear support.

  1. Rail cover
  2. Roof rail
