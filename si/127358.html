Tail gate glass installation (All models)

1. Tail gate glass installation

1. Clean the tail gate glass using the unleaded gasoline.

2. Clean the body panel using the unleaded gasoline.

Note

  • Clean the bonding surface of the tail gate glass. 
  • Wipe away any remaining caulking agent from the body using a clean cloth and unleaded gasoline. 

3. Apply the primer to the body.

Note

  • Apply Sunstar BP-901 or equivalent to the bonding surface on the body side.
  • Primer application width: Top 20 mm {0.79 in} or more, Side 20 mm {0.79 in} or more, Bottom 28 mm {1.1 in} or more

4. Install the spacer to the body.

5. Install the retainer to the tail gate glass.

Note

  • Use a new spacer and retainer. 

6. Install the tail gate finisher to the panel.

Note

  • Use a new tail gate finisher.

  1. Retainer
  2. Tail gate finisher
  3. Spacer
  4. 0.3 mm {0.01 in}

7. Apply the primer to the tail gate glass.

Note

  • Apply Sunstar GP-402 or equivalent to the bonding surface on the rear window side.
  • Apply the primer as described in the figure.

  1. Sealing agent
  2. Primer application location
  3. 10 mm {0.39 in}
  4. 8 mm {0.24 in}
  5. 10 mm {0.39 in}
  6. 4 mm {0.16 in}
  7. 47 mm {1.85 in}
  8. 41 mm {1.61 in}
  9. 15 mm {0.59 in}
  10. 9 mm {0.35 in}
  11. 28 mm {1.10 in}
  12. 21 mm {0.83 in}

8. Apply the sealing adhesive to the tail gate glass.

Note

  • When using an air spray gun, air pressure must be kept lower than the specified value.

Specified air pressure： 147 to 294 kPa { 1.5 to 3.0 kgf/cm2 / 21 to 43 psi } Or less

Note

  • After the primer completely dries, apply the sealing agent, Sunstar penguinseal #560 or equivalent, along the edge of the tail gate glass so that the sealing agent forms a bonding area of 20 mm {0.79 in} at the left lower section of the tail gate glass.

  1. Spacer
  2. Tail gate glass support
  3. Sealing agent
  4. Retainer
  5. 20 mm {0.79 in}

Caution

  • After applying the primer, leave it for at least 1 minute for air drying.
  • Bonding must be accomplished within 5 minutes after the sealing agent is applied.

9. Install the tail gate glass to the panel.

Note

  • Insert the tail gate glass support into the panel and press in the panel side to fasten.

  1. Mounting hole
  2. Tail gate glass support

Note

  • Confirm that there is no water leakage from the tail gate window.

10. Install the tail gate finisher to the tail gate.

11. Connect the rear defogger connector to the tail gate glass.
