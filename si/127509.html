BCM removal (All models)

1. BCM safety information

Caution

  • When replacing the BCM make sure to write the vehicle information to the new BCM.
  • When replacing BCM, before removal upload the old BCM data to a scan tool, referring to the following.
  • If communication cannot be made with a scan tool, remove it with no change.
  • When replacing the BCM, use only one scan tool from start to end of the replacement operation. Correct information cannot be written if the scan tool is changed　during replacement.

1. Check specification of vehicle.

Note

  • The following procedure is required when replacing the BCM.

2. BCM upload

Note

  • The following procedure is required when replacing the BCM.
  • When communication between the BCM and a scan tool cannot be carried out, do not perform the above-mentioned operation.

1. Connect a scan tool to the DLC.

2. Turn ON the ignition switch.

3. Select the Diagnostics.

4. Select the body.

5. Select the BCM.

6. Select Programming & Resetting.

7. Select BCM Data Upload.

Note

  • Upload the vehicle data to a scan tool by following the on-screen instructions.

3. Battery ground cable disconnect

1. Disconnect the battery ground cable from the battery.

Caution

  • Do not disconnect within 1 minute after turning OFF the ignition switch.

4. Dash side trim pad removal

1. Remove the dash side trim pad from the body.

Note

  • The following applies to RHD models.

  1. Dash side trim pad
  2. Retainer

Note

  • The following applies to LHD models.

  1. Dash side trim pad
  2. Retainer

5. BCM removal

1. Disconnect the harness connector from the BCM.

Note

  • Remove it together with the bracket installed onto the dash panel.

2. Remove the BCM from the dash panel.

Note

  • The following applies to RHD models.

  1. BCM
  2. Bolt

Note

  • The following applies to LHD models.

  1. Bolt
  2. BCM
