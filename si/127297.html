Tail gate reassembly (All models)

1. Rear camera installation

Note

  • The following applies to models with a rear camera.

1. Install the rear camera to the tail gate.

Note

  • Connect the rear camera harness connector.
  • Install the rear camera, and tighten the screw.

  1. Grommet
  2. Screw
  3. Rear camera

2. Dove tail installation

1. Install the dove tail to the tail gate.

Tightening torque： 5.7 N・m { 0.58 kgf・m / 50 lb・in }

  1. Bolt
  2. Dovetail

3. Tail gate hinge installation

1. Install the tail gate hinge to the tail gate.

Note

  • Tighten the bolt to the specified torque.

Tightening torque： 34.3 N・m { 3.5 kgf・m / 25 lb・ft }

  1. Tail gate hinge
  2. Bolt

4. Tail stoplight installation

1. Install the tail stoplight to the tail gate.

Note

  • Install the clip and tighten the 3 nuts to install the tail stoplight to the tail gate.

  1. Nut
  2. Cap nut
  3. Taillight
  4. Clip

2. Connect the connector to the socket.

3. Install the cover to the tail gate.

5. Tail gate lock installation

1. Install the tail gate lock to the tail gate.

Note

  • Install the fastening bolt and connect the connector.

Tightening torque： 5.9 to 7.8 N・m { 0.6 to 0.8 kgf・m / 52 to 69 lb・in }

  1. Tail gate lock
  2. Bolt

6. Outside handle installation

1. Install the outside handle to the tail gate.

Note

  • Install the fastening bolt and tighten to the specified torque.

Tightening torque： 6.5 N・m { 0.66 kgf・m / 58 lb・in }

Note

  • Connect the harness connector.

  1. Outside handle
  2. Bolt

7. Tail gate garnish installation

1. Install the tail gate garnish to the tail gate.

Note

  • Install the 2 nuts and the 4 clips.

Tightening torque： 1.8 N・m { 0.18 kgf・m / 16 lb・in } Nut

  1. Nut
  2. Stud bolt
  3. Clip

8. Rear wiper motor installation

1. Install the wiper motor to the tail gate.

Note

  • Install the wiper motor and tighten the bolt.

Tightening torque： 8 N・m { 0.8 kgf・m / 71 lb・in }

  1. Bolt
  2. Harness connector

2. Install the grommet to the rear window.

Note

  • Align the grommet marks so they face upward and install as shown in the diagram.
  • When installing, soapy water can be applied to the grommet fitting section.
  • There must be no clearance between the grommet and rear window.

  1. 20°
  2. Grommet fitting section
  3. Grommet

  1. Bolt
  2. Harness connector

Tightening torque： 8 N・m { 0.8 kgf・m / 6 lb・ft }

Note

  • Install the tail gate trim.

9. Rear wiper arm installation

1. Install the wiper arm to vehicle.

Note

  • Install the blade to the position shown in the following illustration.

Caution

  • When tightening the nut, hold down the arm in order to prevent it from springing up.

Tightening torque： 8 N・m { 0.8 kgf・m / 71 lb・in }

Note

  • Steady the wiper blade by moving it up and down on the glass surface.

  1. 13 mm (0.51 in)
  2. 8 mm (0.31 in)
  3. Ceramic paint line

10. Tail gate trim cover installation

1. Install the tail gate trim cover to the tail gate.

Note

  • Insert the 13 clips for the tail gate trim cover.
  • Install the screw for the assist grip to the cab.

  1. Clip position
  2. Screw
