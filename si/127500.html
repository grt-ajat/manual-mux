Tailgate opener switch installation (All models)

1. Tailgate opener switch installation

1. Install the tailgate opener switch to the outside handle.

Note

  • Tighten the 2 screws.

  1. Outside handle
  2. Tailgate opener switch
  3. Screw

2. Outside handle installation

1. Install the outside handle to the tail gate.

Note

  • Install the fastening bolt and tighten to the specified torque.

Tightening torque： 6.5 N・m { 0.66 kgf・m / 58 lb・in }

Note

  • Connect the harness connector.

  1. Outside handle
  2. Bolt

3. Tail gate trim cover installation

1. Install the tail gate trim cover to the tail gate.

Note

  • Insert the 13 clips for the tail gate trim cover.
  • Install the screw for the assist grip to the cab.

  1. Clip position
  2. Screw
