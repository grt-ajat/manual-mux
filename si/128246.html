ATF cooler pipe installation (TB-50LS)

1. ATF cooler pipe cleaning

1. Clean the ATF cooler pipe using cleaning fluid.

Note

  • Clean the ATF Cooler and pipes using a cleaning agent and compressed air.

Compressed air： 200 kPa { 2.0 kgf/cm2 / 29 psi } or less

Note

  • After air-drying the cleaning agent in the ATF Cooler and pipe, clean them with transmission fluid.

2. ATF hose installation

1. Install the ATF hose to the oil pipe.

Note

  • Shove the ATF hoses securely.

  1. Shove the ATF hose completely over the hose fitting end.
  2. Shove the ATF hose completely over the corner end of the hose fitting.

3. ATF cooler pipe installation

1. Temporarily tighten the ATF cooler pipe to the transmission.

Note

  • Temporarily install the ATF cooler pipe and clips, determine the assembly direction of the ATF cooler pipe.

2. Securely tighten the ATF cooler pipe to the transmission.

Note

  • Tighten the ATF cooler pipe and clips at the specified torque.

Tightening torque： 23 N・m { 2.3 kgf・m / 17 lb・ft } Bolt

Tightening torque： 44 N・m { 4.5 kgf・m / 32 lb・ft } Flare nut

3. Connect the ATF cooler pipe to the ATF hose.

Note

  • Connect the ATF cooler pipe to the ATF hoses and fix it with the clip.

4. Drain plug installation

1. Clean the drain plug using cleaning fluid.

2. Install the drain plug to the oil pan.

Note

  • Install a new gasket and the drain plug to the oil pan.

Caution

  • Be sure to install the gasket in the correct direction as shown in the figure.

Tightening torque： 20 N・m { 2.0 kgf・m / 15 lb・ft }

  1. Gasket
  2. Drain Plug

5. ATF filling

1. Replenish the transmission with the ATF.

Caution

  • When Replenishing ATF, always perform the oil level adjustment using the overflow hole.
