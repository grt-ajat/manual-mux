ABS works frequently but the brake is not effective or TCS operates frequently (All models)

1. ABS works frequently but the brake is not effective or TCS operates frequently description of symptom

ABS works frequently but the brake is not effective or TCS operates frequently.

2. ABS works frequently but the brake is not effective or TCS operates frequently diagnostics

Note

  • The following is the diagnosis when not using a scan tool.

1. Inspect the tire.

2. If a problem is not found, replace the tire.

3. Inspect the front and rear brake distribution of the vehicle.

4. If a problem is found, repair or replace the relevant area.

5. Inspect assembly condition of parts related to the axle.

6. If a problem is found, repair or replace the relevant area.

7. Inspect the installation condition of the wheel speed sensor.

8. If a problem is found, repair the installation condition of the wheel speed sensor.

9. Inspect the wheel speed sensor.

Note

  • There should be no foreign matter attached on the wheel speed sensors.

10. Inspect the sensor rotor.

Note

  • There should be no foreign matter attached on the sensor rotor.

11. Inspect the wheel speed sensor circuit.

12. Replace the EHCU.

Refer to "4.Brakes 4D.ABS(All models) EHCU removal".

Refer to "4.Brakes 4D.ABS(All models) EHCU installation".

13. Set the vehicle information on the EHCU.

Refer to "4.Brakes 4D.ABS(All models) EHCU setting".

Note

  • The following is the diagnosis when using a scan tool.

14. Turn OFF the ignition switch.

15. Connect a scan tool to the DLC.

16. Check the wheel speed display using the scan tool.

Note

  • Perform the procedure for each wheel.

Confirm whether the lowest speed of all the wheel speed displays are at or below 4 km/h (2 mph) and stable.

17. If a problem is found, replace the applicable wheel speed sensor.

Refer to "4.Brakes 4D.ABS(All models) front wheel speed sensor removal".

Refer to "4.Brakes 4D.ABS(All models) front wheel speed sensor installation".

Refer to "4.Brakes 4D.ABS(All models) rear wheel speed sensor removal".

Refer to "4.Brakes 4D.ABS(All models) rear wheel speed sensor installation".

18. Perform diagnosis for when not using a scan tool.

19. Restore the vehicle.

20. Perform a test-run.

21. Check for a DTC using a scan tool.

Note

  • If any DTCs are set, inspect the applicable DTC.
