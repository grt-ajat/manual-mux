Diagnostic system check-engine controls (4JJ1)

1. Diagnostic system check-engine controls description of function

The diagnostic system check-engine controls are a sequenced method for checking problems caused by the functional failures in the engine control system, and are also the starting point for all diagnostics for drivability complaints. By correctly using these diagnostic steps, it is possible to reduce the time for diagnostics and prevent replacing parts that are not defective.

2. Diagnostic system check-engine controls

Note

  • If there are no complaints regarding drivability, do not perform these diagnostic steps unless otherwise instructed to do so.
  • Check the Service Bulletin before starting the diagnosis.
  • Do not clear the DTCs unless it is instructed in a diagnostic step.
  • If a malfunction is found in the engine starting system, inspect the starting system.
  • The battery should be fully charged.
  • The battery cable should be normal and securely connected.
  • The ECM ground should be securely connected to the correct position.
  • The ECM harness connector should be clean and correctly connected.
  • The ECM terminals should be clean and correctly connected.
  • The injector ID code should be programmed correctly.
  • The immobilizer information should be programmed correctly.
  • If fuel system DTCs (P0087, P0088, P0089, P0093, or P1093) or other DTCs are set, diagnose sensor DTCs, solenoid DTCs, actuator DTCs and relay DTCs first.

1. Inspection of scan tool power up

Install a scan tool.

If the scan tool powers up, go to Inspection of scan tool communication.

If the scan tool does not power up, refer to Scan tool does not power up, and repair as necessary.

2. Inspection of scan tool communication

Turn ON the ignition switch.

Verify the vehicle specifications and the status of communication with the following control units using a scan tool.

ECM
TCM (A/T TB-50LS only)
TCCM (4WD model)
EHCU (if so equipped)
ICU (if so equipped)
SRS control unit (if so equipped)
BCM (if so equipped)
Instrument panel cluster
PESS control unit (if so equipped)

If communication with all the control modules can be established, go to Engine cranking verification.

If communication cannot be established with any of the control modules, refer to Scan tool does not communicate with CAN device, and repair as necessary.

3. Engine cranking verification

Caution

  • If the immobilizer system is activated, the ECM deactivates the fuel injection and performs starter cut to cancel cranking.

Crank the engine.

If the engine does not crank, go to DTC verification 1.

If the engine cranks, go to Engine start verification.

4. Engine start verification

Start the engine.

If the engine does not start, refer to Starting system check, and repair as necessary.

If the engine starts, go to DTC verification 2.

5. DTC verification 1

Observe the ECM DTC information with a scan tool.

If DTCs P0633, P161B, or U0167 are set, diagnose the applicable DTCs.

If the above DTCs are not set, inspect starting system parts and circuits, and repair as necessary.

6. DTC verification 2

Verify the following control module DTCs with a scan tool.

ECM
PESS control unit (if so equipped)
ICU (if so equipped)
TCM (A/T TB-50LS only)

If DTCs beginning with U or DTCs related to communication with other control modules are set, diagnose the applicable DTCs.

If ECM DTCs P0601, P0602, P0604, P0606, P060B, or P1621 are set, diagnose the applicable DTCs.

If ECM DTC P0563 is set, diagnose the applicable DTC.

If any DTCs are set, other than the above DTC, go to the applicable DTC diagnosis.

7. Confirmation 1 of symptom

Verify the customer's concern is related with the following systems.

Automatic transmission
Immobilizer

If it is applicable, refer to Diagnostic system check for the relevant system, and inspect it.

8. Confirmation 2 of symptom

Ensure the following symptoms are met.

Hard start
Rough, unstable
High idle speed
Cuts out
Surges
Lack of power, sluggishness, or sponginess
Hesitation, sag, or stumble
Abnormal combustion noise
Poor fuel economy
Black smoke
White smoke

If a problem is found, repair the applicable symptom.

If any of the symptoms are not met, refer to Intermittent conditions of engine, and inspect.
