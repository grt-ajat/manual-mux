DTC P0217 (Flash Code 542) Engine Coolant Over Temperature Condition (4JK1)

1. DTC P0217 diagnostics

Note

  • After starting the engine, the ECT should rise to about 80 to 85°C (176 to 185°F) then stabilize when the thermostat opens.
  • The Number of Engine Coolant Overtemperature Events display on the scan tool indicates the number of overheats.

1. Diagnostic system check

Refer to "202.Description Engine 15C.Functional Inspection(4JK1) diagnostic system check-engine controls".

2. Prioritized DTC

If DTC P0117 is set at the same time, go to the diagnosis for DTC P0117.

3. Inspection of engine cooling system

Inspect the engine cooling system for the following conditions, and repair as necessary. Go to Repair verification.

Insufficient coolant level
Coolant leakage

Refer to "1.Engine 1D.Cooling(4JK1) coolant inspection".

Cooling fan belt slip

Refer to "1.Engine 1D.Cooling(4JK1) cooling fan belt inspection".

Cooling fan clutch malfunction

Refer to "1.Engine 1D.Cooling(4JK1) cooling fan clutch inspection".

Thermostat malfunction

Refer to "1.Engine 1D.Cooling(4JK1) thermostat inspection".

Water pump malfunction

Refer to "1.Engine 1D.Cooling(4JK1) water pump assembly inspection".

Radiator clogging

Refer to "1.Engine 1D.Cooling(4JK1) radiator inspection".

If it is normal, go to Water temperature verification.

4. Water temperature verification

While observing the Engine Coolant Temperature (ECT) display on the scan tool, completely warm up the engine.

Standard： 110 ℃ { 230 °F }

If the reading is less than the specified value, ask the driver if the overheat has occurred due to low engine coolant level, etc. If engine overheat has occurred in the past, make sure to inspect the engine, and repair as necessary. Go to Repair verification.

If the reading is more than or equal to the specified value, inspect the engine coolant temperature sensor, and repair as necessary. Go to Repair verification.

Refer to "1.Engine 1D.Cooling(4JK1) engine coolant temperature sensor inspection".

2. DTC P0217 repair verification

Clear the DTC with a scan tool.

Turn OFF the ignition switch for at least 30 seconds.

While observing the Engine Coolant Temperature (ECT) display on the scan tool, completely warm up the engine.

Standard： 110 ℃ { 230 °F }

If the reading is more than or equal to the specified value, go to Prioritized DTC.

If the reading is less than the specified value, observe the DTC information with a scan tool.
