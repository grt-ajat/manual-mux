Tongue side 2nd seat belt installation (All models)

1. Tongue side 2nd seat belt installation

1. Install the tongue side 2nd seat belt to the panel.

Tightening torque： 46.0 N・m { 4.7 kgf・m / 34 lb・ft }

2. Install the seat belt upper anchor bolt to the panel.

Tightening torque： 46.0 N・m { 4.7 kgf・m / 34 lb・ft }

  1. Seat belt upper anchor bolt
  2. Tongue side 2ND seat belt
  3. Seat belt lower anchor bolt

2. Upper quarter trim cover installation

1. Install upper quarter trim cover to the quarter panel.

Note

  • Tighten the screw and install the cover.

2. Install the bezel to the quarter panel.

Note

  • Insert the seat belt through the opening in the bezel.

3. Install the seat belt lower anchor bolt to the floor.

Note

  • Install the 2nd and 3rd seat belt lower anchor bolts

Tightening torque： 46 N・m { 4.7 kgf・m / 34 lb・ft }

4. Install the assist grip to the quarter panel.

Tightening torque： 7 N・m { 0.7 kgf・m / 62 lb・in }

  1. Upper quarter trim cover
  2. Screw
  3. Cover
  4. Bezel
  5. Screw
  6. Assist grip
  7. Cover (For vehicle with curtain airbag)
  8. Screw (For vehicle with curtain airbag)
  9. Cover
  10. Screw
  11. Bezel

3. Lower quarter trim cover installation

1. Install the lower quarter trim cover to the quarter panel.

Note

  • Tighten the screw and install the cover.

2. Install the luggage hook to the quarter panel.

Tightening torque： 8 N・m { 0.8 kgf・m / 6 lb・ft }

  1. Luggage hook
  2. Screw
  3. Cover
  4. Cover
  5. Screw
  6. Lower quarter trim cover

3. Install the accessory socket to the lower quarter trim cover.

Note

  • This procedure is applicable to the vehicle with the accessory socket.

  1. Cover
  2. Accessory socket

4. Luggage rear trim installation

1. Install the luggage rear trim to the floor.

  1. Luggage rear trim

5. Tail gate weather strip installation

1. Install the buffer to the body.

Note

  • Install the buffer before installing the weather strip to the lower notch of the flange body while being cautious not to damage the painted surfaces of vehicle.

  1. Buffer
  2. Weather strip

2. Install the tail gate weather strip to vehicle.

  1. Tail gate weather strip

1. Install the rear door finisher to the body.

  1. Rear door finisher

7. Sill plate installation

1. Install the sill plate to the floor.

Note

  • Install the rear sill plate.
