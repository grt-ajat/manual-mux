Diagnostic system check-body controls (BCM with PESS)

1. Diagnostic system check-body controls description of function

The diagnostic system check - body controls is an organized approach to identifying a problem caused by a BCM malfunction, and must be the starting point for any drivability complaint diagnosis. Using it correctly will reduce diagnostic time and prevent replacement of good parts.

2. Diagnostic system check-body controls

Note

  • If there are no complaints regarding drivability, do not perform these diagnostic steps unless otherwise instructed to do so.
  • Check the service technical report before starting the diagnosis.
  • Do not clear the DTCs unless it is instructed in a diagnostic step.
  • The battery should be fully charged.
  • The battery cable should be normal and securely connected.
  • The BCM ground should be securely connected to the correct position.
  • The BCM harness connector should be clean and correctly connected.
  • The BCM terminal should be clean and matched correctly.
  • Vehicle information should be programmed correctly in the BCM.

1. Inspection of scan tool power up

Install a scan tool.

Verify whether the scan tool powers up.

If it does not power up, refer to Scan tool does not power up in the Description Engine section, and repair as necessary.

2. Inspection of scan tool communication

Install a scan tool.

Turn ON the ignition switch.

Verify the vehicle specifications and the status of communication with the following control units with the scan tool.

ECM
TCM (A/T TB-50LS only)
TCCM (4WD model)
EHCU
ICU
SRS control unit (if so equipped)
BCM
Instrument panel cluster
PESS control unit

If communication can be established with all control modules, go to DTC verification.

If there are any control modules with which communication cannot be established, refer to Scan tool does not communicate with CAN device in the Description Engine section, and repair as necessary.

3. DTC verification

Observe the BCM DTC information with a scan tool.

If a DTC is set, diagnose the applicable DTC.

4. Symptom verification

Check if the contents of claims from customers are related to the Passive Entry and Start System.

If applicable, refer to Diagnostic system check - passive entry and start system, and inspect as necessary.

Verify that the customer's concern is related with the immobilizer system.

If applicable, refer to Diagnostic system check - immobilizer controls, and inspect as necessary.

Operate the vehicle and check if the following conditions are met.

All doors neither lock nor unlock
All of the doors do not lock or unlock
Door at the driver side does not lock or unlock
Door at the passenger side does not lock or unlock
Rear right door does not lock or unlock
Rear left door does not lock or unlock
Door lock at the driver side does not work
Door lock works with keyless entry functions, but does not answer back, or room light does not light
Rear wiper does not operate
Wiper motor does not stop
Rear washer motor does not operate

If a problem is found, repair the applicable symptoms.

If any of the symptoms are not met, refer to Intermittent conditions of body control system, and inspect as necessary.
