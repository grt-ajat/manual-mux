Keyless entry system check (BCM with Keyless Entry)

1. Keyless entry system check description of function

The keyless entry system functions as a vehicle entry device. Keyless entry locks or unlocks the door when the corresponding button of the remote controller key is pressed. It is operated by sending radio waves from the transmitter to the receiver unit. The receiver unit detects the signal, and then performs the requested function by sending serial data messages to the BCM. The system may malfunction due to low battery of the transmitter or the vehicle, or radio wave disturbance from aftermarket devices such as power inverters or computers. The system may also malfunction due to radio wave disturbance in high-traffic areas.
The keyless entry system has the following functions.

Lock and unlock functions by remote controller operation
Remote controller key registration function
Answer-back function
Interior light ON/OFF interlocking function

2. Keyless entry system check

1. Diagnostic system check

Refer to "223.Description Body Control 36C.Functional Inspection(BCM with Keyless Entry) diagnostic system check-body controls".

2. DTC verification

If DTCs B0970, B2555, B3100, B3130, or B3135 are set, diagnose the applicable DTCs.

3. Inspection of keyless entry system operation

Using a remote controller key, command the doors to lock and unlock.

If it does not lock or unlock, go to Inspection of transmitter.

If it locks and unlocks but does not answer back, go to Circuit diagnosis 4.

If the dome light does not operate when unlocked, refer to Operation abnormality of the dome light function in the Description Body, Cab, Accessories section, and inspect.

Refer to "210.Description Body, Cab, Accessories 23D.Symptom(All models) operation abnormality of the dome light function".

4. Inspection of transmitter

Inspect the transmitter for the following conditions, and repair or replace as necessary. Go to Repair verification.

It should be a correct model that corresponds with the vehicle system.
There should be no physical damage to the key cover and the button.
There should be no battery deterioration.

If it is normal, go to Verification of ID code registration.

5. Verification of ID code registration

Register the remote controller key with a scan tool.

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) BCM setting".

If it is already registered, go to Circuit diagnosis 1.

If the ID code cannot be registered, refer to ID Code not yet programmed or no remote key programmed, and inspect as necessary.

Refer to "223.Description Body Control 36D.Symptom(BCM with Keyless Entry) ID Code not yet programmed or no remote key programmed".

6. Circuit diagnosis 1

Close all doors and the tailgate.

Observe the Door Switch display on the scan tool.

If OPEN is displayed, go to Circuit diagnosis 3.

If CLOSE is displayed, go to Door switch and tailgate switch inspection.

7. Door switch and tailgate switch inspection

Inspect the door switch and the tailgate switch, and repair as necessary. Go to Repair verification.

Refer to "9.Body, Cab, Accessories 9A.Lighting Systems(All models) door switch inspection".

Refer to "9.Body, Cab, Accessories 9A.Lighting Systems(All models) door switch removal".

Refer to "9.Body, Cab, Accessories 9A.Lighting Systems(All models) door switch installation".

Refer to "9.Body, Cab, Accessories 9J.Security and Locks(All models) tail gate lock removal".

Refer to "9.Body, Cab, Accessories 9J.Security and Locks(All models) tail gate lock installation".

Refer to "9.Body, Cab, Accessories 9J.Security and Locks(All models) tail gate lock inspection".

If it is normal, go to Circuit diagnosis 2.

8. Circuit diagnosis 2

Turn OFF the ignition switch.

Install the door switch and the tailgate switch.

Disconnect the BCM harness connector.

Close all doors and the tailgate.

Connect a test lamp between the door switch signal circuit and the power supply circuit.

If the test lamp does not illuminate, go to BCM replacement.

If the test lamp illuminates, repair the short to ground in the door switch and tailgate switch signal circuit. Go to Repair verification.

9. Circuit diagnosis 3

Pull out the remote controller key from the key cylinder.

Disconnect the BCM harness connector.

Connect a test lamp between the key reminder switch signal circuit and the body ground.

If the test lamp illuminates, go to Inspection for short circuit in key reminder switch signal circuit.

If the test lamp does not illuminate, go to BCM replacement.

10. Inspection for short circuit in key reminder switch signal circuit

Inspect for a short to the power supply circuit in the key reminder switch signal circuit, and repair as necessary. Go to Repair verification.

If the circuit is normal, replace the ignition switch. Go to Repair verification.

Refer to "9.Body, Cab, Accessories 9C.Electrical Control, Entertainment(All models) ignition switch removal".

Refer to "9.Body, Cab, Accessories 9C.Electrical Control, Entertainment(All models) ignition switch installation".

11. Circuit diagnosis 4

Disconnect the BCM harness connector.

Connect a fused jumper wire between the answer-back control circuit and the body ground.

If the hazard lamp flashes, go to Inspection of BCM harness connector.

If the hazard lamp does not flash, repair an open circuit or high resistance in the answer-back control circuit. Go to Repair verification.

12. Inspection of BCM harness connector

Inspect the connection status of the BCM harness connector, and repair as necessary. Go to Repair verification.

If the connection status is normal, go to BCM replacement.

13. BCM replacement

Replace the BCM. Go to Repair verification.

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) BCM removal".

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) BCM installation".

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) BCM setting".

14. Repair verification

Reconnect all of the disconnected fuses and harness connectors.

Close all doors and the tailgate.

Using a remote controller key, command the doors to lock and unlock.

If it does not operate properly, go to Inspection of keyless entry system operation.
