Washer motor installation (All models)

1. Washer motor installation

1. Install the washer motor to the washer tank.

Note

  • Replace the grommet with a new one, and set it to the washer tank side.
  • Insert the water supply port of the motor into the grommet.
  • The following applies to models without a rear wiper.

  1. Washer motor

Note

  • The following applies to models with a rear wiper.

  1. Washer motor, rear wiper side
  2. Washer motor, windshield wiper side

2. Washer tank installation

1. Install the washer tank to vehicle.

Note

  • The following applies to models with a rear wiper.
  • Connect the connector to the washer motor of the rear wiper.

  1. Connector

Tightening torque： 6.5 N・m { 0.7 kgf・m / 58 lb・in }

  1. Upper side tab
  2. Washer tank
  3. Bolt
  4. Lower side tab

2. Connect the harness connector to the washer motor.

Note

  • Connect the washer motor of the windshield wiper.

  1. Washer motor
  2. Washer tank
  3. Harness connector

3. Tank joint hose installation

1. Install the tank joint hose to the washer tank.

Tightening torque： 5.9 N・m { 0.6 kgf・m / 52 lb・in }

2. Install the clip to the tank joint hose.

3. Connect the washer hose to the clip.

  1. Connection of washer hose
  2. Clip
  3. Bolt

4. Inner liner installation

1. Install the inner liner to the front fender panel.

Note

  • Install the fixing screw.

5. Mudguard installation

1. Install the mud guard to the front fender panel.

6. Battery ground cable connect

1. Connect the battery ground cable to the battery.
