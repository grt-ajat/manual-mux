Abnormal combustion noise (4JJ1)

1. Abnormal combustion noise description of symptom

Engine knocking occurs many times. Usually, it becomes particularly notable when accelerating. A sharp metallic sound is produced when the throttle opening angle is changed.

2. Abnormal combustion noise diagnostics

Note

  • Preliminary inspection

Inspect the engine control system.
Check if any problem exists in the vehicle.
Inspect whether smoke is present with combustion noise.
Inspect the fuel quality.
Verify the injector ID code of each cylinder.
Check the scan tool data list.
Check for related Service Bulletins.

Note

  • Sensor inspection

Compare each display for Engine Coolant Temperature (ECT), Intake Air Temperature (IAT), and Fuel Temperature with the scan tool data list when the engine is stopped and cold. If the difference between the temperature values is 5 °C (9 °F) or more when the engine has cooled, inspect for high resistance in each circuit and for sensor failure.
{When the ignition switch is turned ON, the MAF sensor is heated, and as a result, the IAT shows a higher value than the normal intake air temperature.}
Observe the Fuel Rail Pressure (FRP) Sensor display on the scan tool. It is normal if the Fuel Rail Pressure (FRP) Sensor display shows 0.9 - 1.0 V when the engine is not started with the ignition switch turned ON after at least 1 minute from the stop of engine operation. If not, inspect for high resistance in each circuit and for any sensor failure.
Fully depress the accelerator pedal while idling in neutral, and verify the Fuel Rail Pressure (FRP) and Desired Fuel Rail Pressure (FRP) displays. It is normal if the Fuel Rail Pressure (FRP) display and the Desired Fuel Rail Pressure (FRP) display follow each other within a difference of 5 MPa {725 psi}.
The CKP sensor is securely installed, and the sensor rotor is not damaged.

Note

  • Fuel system inspection

If excessive smoke is present, inspect the injector for stuck open. Inspect the combustion chamber for fuel intrusion.
Remove the injector, and visually inspect it.
Perform the cylinder balance test with the scan tool.
Perform the injector forced drive with the scan tool.
Confirm the fuel compensation display of each cylinder with the scan tool while idling.

Note

  • Air intake system inspection

Inspect the air cleaner element for clogging.
Inspect the intake air duct for clogging.

Note

  • Engine mechanical inspection

Inspect the compression pressure of the engine.
Inspect the basic components of the engine including the camshaft, cylinder head, piston, etc.
Inspect the combustion chamber for excessive oil intrusion.

Note

  • Other inspections

Inspect for any other cause that generates similar sound, such as loose components, bracket, mount, deteriorated clutch damper spring.
