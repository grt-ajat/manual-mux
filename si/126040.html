Intercooler installation (4JJ1)

1. Intercooler installation

1. Install the intercooler to the frame.

Tightening torque： 20 N・m { 2.0 kgf・m / 15 lb・ft }

2. Connect the intake air hose to the intercooler.

Note

  • Connect it at both the turbocharger and intake throttle sides.

Tightening torque： 5 N・m { 0.5 kgf・m / 44 lb・in }

  1. Intake hose (turbocharger side)
  2. Intake hose (intake throttle side)

2. Front end engine hood stay installation

1. Install the front end engine hood stay to the frame.

3. Front bumper impact bar assembly installation

1. Install the front bumper impact bar assembly to vehicle.

Note

  • Tighten the 4 fixing bolts to the specified torque.

Tightening torque： 62 N・m { 6.3 kgf・m / 46 lb・ft }

2. Install the front bumper support to vehicle.

Note

  • Install the 6 nuts.

3. Install the bumper seal to vehicle.

Note

  • Install the 6 clips on one side.

  1. Upper bumper seal
  2. Lower bumper seal

4. Front bumper assembly installation

1. Install the front bumper assembly to vehicle.

Note

  • Install the 10 clips.

2. Connect the harness connector to the fog light.

  1. Clip
  2. Front bumper assembly

5. Underguard installation

  2. Clip

3. Install the front underguard to the frame.

Note

  • Tighten the 5 bolts.

  1. Front underguard
  2. Rear underguard
  3. Bolt

Note

  5. Rear underguard
