Front seat installation (All models)

1. Seat slide adjuster installation

1. Install the seat slide adjuster to the front seat.

Note

  • Replace the floor panel insulators with new one, before installing the front seat.
  • Tighten the 4 fixing nuts inside the slide adjuster to the specified torque.
  • Tighten the 3 fixing nuts outside the slide adjuster to the specified torque.

Tightening torque： 21 N・m { 2.1 kgf・m / 15 lb・ft }

  1. Insulator

2. Front seat back installation

1. Install the front seat back to the front seat.

Note

  • Tighten the 4 bolts to the specified torque.

Tightening torque： 46 N・m { 4.7 kgf・m / 34 lb・ft }

Note

  • The following applies to power seats.
  • Install the side cover.
  • Install the switch knob.

  1. Side cover
  2. Side cover
  3. Switch knob

Note

  • The following applies to seats with a seat lifter.
  • Install the side cover.
  • Install the cap and lifter lever.
  • Install the lever knob.

  1. Cap
  2. Lever knob
  3. Side cover
  4. Side cover
  5. Lifter lever

Note

  • The following applies to bucket seats.
  • Install the side cover.
  • Install the lever knob.

  1. Lever knob
  2. Side cover
  3. Side cover

Note

  • Install the headrest.

3. Buckle side passenger seat belt installation

1. Install the passenger seat belt buckle to the passenger seat.

Note

  • Install with the passenger seat belt buckle mounting bolts.

Tightening torque： 40.0 N・m { 4.1 kgf・m / 30 lb・ft }

  1. Passenger seat belt buckle

4. Buckle side driver seat belt installation

1. Install the driver seat belt buckle to the driver seat.

Note

  • Install with the driver seat belt buckle mounting bolts.

Tightening torque： 40 N・m { 4.1 kgf・m / 30 lb・ft }

  1. Driver seat belt buckle

5. Front seat installation

1. Install the front seat to the floor panel.

Note

  • For vehicles with power seat, connect the power seat harness connector.
  • For vehicles with side airbags, connect the side airbag harness connector.
  • Connect the seat belt switch harness connector.

  1. Seat belt switch harness connector
  2. Side airbag harness connector
  3. Power seat harness connector

Note

  • Install the bolt to the specified torque.

Tightening torque： 46 N・m { 4.7 kgf・m / 34 lb・ft }

Note

  • Install the rear cover.

  1. Front seat
  2. Rear cover

6. SRS control unit setting

1. Reactivate the SRS function.

Note

  • Install the SRS fuse to the fuse relay box or connect the battery ground cable to the battery.
  • Turn the ignition switch on, and confirm that the SRS airbag warning light turns off after blinking 7 times.
