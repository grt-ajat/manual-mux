Shock absorber (2WD high ride suspension) (4WD) installation (All models)

1. Shock absorber installation

1. Install the shock absorber to the frame.

Note

  • Install so that the longer stud bolt is installed at the outside of the vehicle.

  1. Long stud bolt

Tightening torque： 55 N・m { 5.6 kgf・m / 41 lb・ft }

2. Temporarily tighten the lower control arm to the frame.

Note

  • Temporarily tighten the nut and cam bolt of the lower control arm.
  • Turn the cam bolt to the position of the setting mark that was put before the disassembly.

3. Apply the oil to the bolt.

Note

  • Apply ISO: VG46 oil or equivalent to the threaded portions of the shock absorber lower side bolt and nut.

4. Install the shock absorber to the lower control arm.

Tightening torque： 152 N・m { 15.5 kgf・m / 112 lb・ft }

Note

  • Tighten the nut and bolt together with the part at the location described in the illustration below.
  • Measure the buffer clearance.

Standard： 26.0 mm { 1.02 in }

  1. Lower control arm
  2. Buffer clearance
  3. Bumper rubber
  4. Center of rear side cam bolt
  5. Trim height

5. Connect the lower ball joint to the knuckle.

Caution

  • Use a new cotter pin.

Tightening torque： 147 N・m { 15.0 kgf・m / 108 lb・ft }

Note

  • Adjust the trim height.

Reference value： 140 mm { 5.512 in }

  1. Lower control arm
  2. Buffer clearance
  3. Bumper rubber
  4. Center of rear side cam bolt
  5. Trim height

6. Install the stabilizer link to the lower control arm.

Tightening torque： 64 N・m { 6.5 kgf・m / 47 lb・ft }

7. Apply the oil to the bolt.

Note

  • Apply ISO: VG46 oil or equivalent to the threaded portions of the cam bolt and nut.

8. Securely tighten the lower control arm to the frame.

Tightening torque： 186 N・m { 19.0 kgf・m / 137 lb・ft }

Note

  • Tighten the nut and bolt together with the part at the location described in the illustration below.
  • Measure the buffer clearance.

Standard： 26.0 mm { 1.02 in }

  1. Lower control arm
  2. Buffer clearance
  3. Bumper rubber
  4. Center of rear side cam bolt
  5. Trim height

2. Tie rod end installation

1. Install the tie rod end to the knuckle.

2. Install the nut to the ball joint.

Tightening torque： 98 N・m { 10.0 kgf・m / 72 lb・ft }

3. Install the cotter pin to the ball joint.

3. Flange installation

Note

  • The following applies to the 4WD vehicles.

1. Install the flange to the front hub.

Note

  • Apply liquid gasket to the flange installation surface and install the flange to the hub.
  • Liquid gasket: LOCTITE 515 or equivalent
  • Install the snap ring and shim.

2. Adjust the shim.

Note

  • Adjust the clearance between the flange and snap ring with the shim.

Standard： 0.0 to 0.2 mm { 0.000 to 0.008 in }

Types of adjust shims
： 0.2 mm { 0.008 in }
： 0.3 mm { 0.012 in }
： 0.5 mm { 0.020 in }
： 1.0 mm { 0.039 in }

4. Hub cap installation

1. Install the hub cap to the front hub.

Note

  • Install the hub cap to the front hub with bolts.

Tightening torque： 59 N・m { 6.0 kgf・m / 44 lb・ft }

5. Disc wheel installation

1. Install the disc wheel to vehicle.

Note

  • Install the wheel nuts in the numbered order.

Tightening torque: 120 N⋅m {12.0 kgf⋅m/87 lb⋅ft}
