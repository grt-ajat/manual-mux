Piston and connecting rod inspection (4JK1)

1. Piston inspection

Note

  • Clean the piston.
  • Carefully remove the carbon from the piston head and piston ring grooves.

Caution

  • Do not use a wire brush to clean the piston, because it causes flaws on the piston.

1. Inspect the piston.

Note

  • Visually inspect for piston cracking, burning, or other excessive wear.
  • Replace the piston if there are abnormalities.

2. Piston measurement

1. Measure the cylinder block using a cylinder gauge.

Note

  • Measure the inner diameter in the thrust direction and the radial direction at the specified position.

Measurement position： 20 mm { 0.79 in } From the top surface of the cylinder block

Note

  • Calculate the average of measured value.

Standard： 95.421 to 95.450 mm { 3.7567 to 3.7579 in }

Limit： 95.48 mm { 3.7590 in }

  1. Measurement position
  2. Measurement direction
  3. Measurement direction

Caution

  • If it exceeds the limit, replace the cylinder block.

2. Measure the piston using the micrometer.

Note

  • Measure the outer diameter of the piston pin at the measurement position.

Measurement position： 11.00 mm { 0.43 in } From the bottom surface of the piston

  1. Measurement position

3. Calculate the clearance from measured value.

Note

  • Calculate the clearance according to the inner diameter of the cylinder block and the outer diameter of the piston.

Standard： 0.042 to 0.080 mm { 0.0017 to 0.0031 in }

Caution

  • Replace the piston if the clearance exceeds the specified value.

4. Select the piston.

Note

  • Grades A, B, and C are marked on the top of the piston.
  • Select a piston so that the clearance is the specified value.
GradeOuter diameter
A： 95.350 to 95.379 mm { 3.7539 to 3.7551 in }
B： 95.360 to 95.389 mm { 3.7543 to 3.7555 in }
C： 95.370 to 95.399 mm { 3.7547 to 3.7559 in }

  1. Grade
  2. Front mark

3. Piston ring measurement

1. Install the piston ring to the cylinder block.

Note

  • Insert it horizontally at the position where it is supposed to be installed to the piston.

Note

  • Push the piston ring into the cylinder until the cylinder bore is at the minimum position.
  • Do not decline the piston ring in either direction.

2. Measure the clearance using a feeler gauge.

Note

  • Measure the piston ring joint clearance.
  • If the measured value exceeds the specified value, replace the piston ring.

Standard： 0.32 to 0.46 mm { 0.0126 to 0.0181 in } 1st piston ring

Standard： 0.42 to 0.66 mm { 0.0165 to 0.0260 in } 2nd piston ring

Standard： 0.27 to 0.56 mm { 0.0106 to 0.0220 in } Oil ring

3. Measure the clearance using a feeler gauge.

Note

  • Measure the clearance between the piston ring grooves and the piston ring.
  • Remove all carbon in the piston ring grooves.
  • Insert the piston ring into the piston ring groove, and measure the clearance.
  • If the clearance between the piston ring groove and the piston ring exceeds the limit, replace the piston or piston ring.

Standard： 0.07 to 0.11 mm { 0.0028 to 0.0043 in } 2nd piston ring

Limit： 0.15 mm { 0.0059 in } 2nd piston ring

Standard： 0.03 to 0.07 mm { 0.0012 to 0.0028 in } Oil ring

Limit： 0.15 mm { 0.0059 in } Oil ring

Note

  • The 1st piston ring cannot be measured.

4. Piston pin inspection

1. Inspect the piston pin.

Note

  • Visually inspect for cracks, scratches, or other damage, and replace the piston pin as necessary.
  • Inspect for a certain degree of resistance when the piston pin is lightly pushed at room temperature.
  • Replace the piston or the piston pin if there is significant looseness or slackening at normal temperature.

5. Piston pin measurement

1. Measure outer diameter using the micrometer.

Note

  • Replace the piston pin if the measured value exceeds the specified value.

Standard： 33.995 to 34.000 mm { 1.3384 to 1.3386 in }

6. Connecting rod assembly measurement

1. Measure inner diameter using a dial gauge.

Note

  • Measure the inner diameter of the connecting rod small end bushing.

2. Calculate the clearance from measured value.

Note

  • Calculate the clearance of the connecting rod small end and the piston pin outer diameter.
  • If the clearance between the bush inner diameter and the pin outer diameter exceeds the limit, replace the connecting rod assembly and the pin.

Standard： 0.008 to 0.020 mm { 0.0003 to 0.0008 in }

Limit： 0.05 mm { 0.0020 in }

3. Apply the engine oil to the piston pin.

4. Install the piston pin to the piston.

5. Turn the piston pin.

Note

  • If the pin rotates stably and smoothly, the clearance is normal.
  • If there is unstableness, measure the clearance.
  • Replace the piston or piston pin if the clearance exceeds the specified value.

Standard： 0.006 to 0.019 mm { 0.0002 to 0.0007 in }

6. Measure the connecting rod assembly using the connecting rod aligner.

Note

  • For connecting rod alignment 100 mm {3.94 in}, measure the twisting and degree of parallelization of the large end hole and small end hole.
  • Replace if the measured value exceeds the specified value.

Standard： 0.05 mm { 0.002 in } Twisting

Standard： 0.05 mm { 0.002 in } Degree of parallelization

7. Connecting rod bearing measurement

1. Remove the connecting rod bearing cap from the connecting rod.

Note

  • The removed caps should be lined up in the order of the cylinder number.
  • Clean the bearing and the crankshaft pin.

2. Inspect the connecting rod bearing.

Note

  • Replace the bearings in sets if there is damage or excessive wear.

3. Put the PLASTIGAUGE on the crankshaft.

Note

  • Place the PLASTIGAUGE on the crankshaft pin.

4. Install the connecting rod bearing cap to the connecting rod.

5. Apply the engine oil to the bolt.

6. Tighten the bolt using a torque wrench.

Note

  • Tighten in numbered order as shown in the diagram.

Tightening torque： 29.4 N・m { 3.0 kgf・m }

Note

  • Tighten the bolts using a special tool.


SST: 5-8840-0266-0 - angle gauge

Specified angle： 45 to 60 °

7. Remove the connecting rod bearing cap from the connecting rod.

8. Measure the clearance using the PLASTIGAUGE.

Note

  • Measure the widest point of the PLASTIGAUGE to find the pin oil clearance.
  • Replace the bearings in sets if the clearance exceeds the specified value.

Standard： 0.033 to 0.075 mm { 0.0013 to 0.0030 in }
