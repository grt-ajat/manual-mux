A/C compressor assembly installation (All models)

1. A/C compressor assembly installation

1. Install the A/C compressor assembly to the A/C compressor bracket.

Tightening torque： 51 N・m { 5.2 kgf・m / 38 lb・ft }

2. Connect the harness connector to the A/C compressor assembly.

  1. Air intake hose
  2. A/C compressor assembly
  3. A/C compressor drive belt

2. A/C pipe installation

1. Install the A/C pipe to the A/C compressor assembly.

Note

  • Do not reuse the O-ring.
  • Always replace them with new ones.
  • Always connect the hose after applying the new O-ring with compressor oil.
  • Tighten the installation bolt for the A/C pipe to the specified torque.

Tightening torque： 15 N・m { 1.5 kgf・m / 11 lb・ft }

3. A/C compressor drive belt installation

1. Install the A/C compressor drive belt to the A/C compressor assembly.

4. A/C compressor drive belt adjustment

1. Adjust tension to the specified value using a tension meter.

Note

  • Loosen the lock nut and adjust bolt of the A/C compressor drive belt tension pulley to free the belt.
  • Turn the adjust bolt for the tension pulley and adjust the belt.
  • Press the specified position on the belt with the specified load to inspect the amount of belt flex.

Load： 98 N { 10 kg / 22 lb }

  1. A/C compressor drive belt measuring section

Note

  • Whether the precise tension has been set can be checked with a sonic tension meter.
A/C compressor drive belt

If determining by flex amountIf using a sonic tension meter

Amount of flexFrequency of vibrations
For a new product： 12.4 to 16.4 mm { 0.488 to 0.646 in }： 92 to 112 Hz
When readjusting tension： 16.5 to 19.1 mm { 0.650 to 0.752 in }： 80 to 92 Hz

Note

  • After adjusting, tighten the tension pulley lock nut to the specified torque.

Tightening torque： 41 N・m { 4.2 kgf・m / 30 lb・ft }

5. Intake hose installation

1. Install the intake hose to the intercooler.

Tightening torque： 5 N・m { 1 kgf・m / 44 lb・in }

Note

  • Install the band clips at 1 location.
  • Align the marks on the pipe side and the hose side, and install.

  1. Intake hose
  2. A/C compressor assembly
  3. A/C compressor belt

6. Refrigerant preparation

1. Fill refrigerant.

Caution

  • For information on the refilling of refrigerant, refer to the instruction manual for the refrigerant recovery equipment.
