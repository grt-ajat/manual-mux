2. Clutch balancer No.3 installation

1. Install the clutch balancer No.3 to the reverse clutch piston sub-assembly.

Note

  • Coat the reverse clutch drum sub-assembly and the clutch balancer No.3 with ATF.
  • Install the reverse clutch return spring sub-assembly and the clutch balancer No.3 to the reverse clutch piston sub-assembly.

Caution

  • Be careful not to damage the O-ring.

2. Install the shaft snap ring to the reverse clutch drum sub-assembly.

Note

  • Place spring compressor on the clutch balancer No.3 and compress the reverse clutch return spring sub-assembly with a press.


SST: 5-8840-2971-0 - Spring compressor

Note

  • Prepare an appropriate pedestal so that the reverse clutch drum sub-assembly is not lowered.
