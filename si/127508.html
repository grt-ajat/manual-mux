BCM installation (All models)

1. BCM installation

1. Install the BCM to vehicle.

Tightening torque： 6.5 N・m { 0.7 kgf・m / 58 lb・in }

2. Connect the harness connector to the BCM.

Note

  • The following applies to RHD models.

  1. BCM
  2. Bolt

Note

  • The following applies to LHD models.

  1. Bolt
  2. BCM

2. Dash side trim pad installation

1. Install the dash side trim pad to the body.

Note

  • The following applies to RHD models.

  1. Dash side trim pad
  2. Retainer

Note

  • The following applies to LHD models.

  1. Dash side trim pad
  2. Retainer

3. Battery ground cable connect

1. Connect the battery ground cable to the battery.

Caution

  • When replacing the BCM make sure to write the vehicle information to the new BCM.
