Tail gate setting (All models)

1. Tail gate hinge adjustment

1. Adjust the tail gate.

Note

  • Move the tail gate hinge and adjust the positions of the body and tail gate.
  • Lightly tap the tail gate hinge with a rubber hammer and adjust the clearance and height.
  • Check the clearance between the roof and the tail gate.

Clearance： 10.7 mm { 0.42 in }

Height： -2.0 mm { -0.08 in }

Note

  • Check the positions of the rear quarter panel and the tail gate.

Clearance： 5.0 mm { 0.197 in }

Height: Same

Note

  • Check the clearance between the rear bumper and the tail gate.

Clearance： 6.5 mm { 0.26 in }

Note

  • Tighten the hinge fastening bolt to the specified torque.

Tightening torque： 40 N・m { 4.1 kgf・m / 30 lb・ft }

  1. Clearance between the roof and the tail gate
  2. Positions of the rear quarter panel and the tail gate
  3. Clearance between the rear bumper and the tail gate

2. Dove tail adjustment

1. Adjust the dove tail.

Note

  • Use the bolt to adjust the movable section of the dove tail to a position where it is securely attached.
  • Align the tailgate and movable section as shown in the following figure, then tighten the bolt after full engagement is made.

Tightening torque： 4 N・m { 0.41 kgf・m / 35 lb・in }
