Starter motor reassembly (4JK1)

1. Magnetic switch reassembly

1. Install the dust cover to the magnetic switch.

2. Install the plunger to the magnetic switch.

3. Install the torsion spring to the magnetic switch.

Note

  • Install it to the magnetic switch hole.

4. Install the shift lever to the magnetic switch.

Note

  • Install it to the plunger hole.

2. Magnetic switch installation

1. Install the magnetic switch to the gear case.

2. Install the dust cover to the gear case.

Tightening torque： 8 N・m { 0.8 kgf・m / 71 lb・in }

3. Pinion assembly installation

1. Assemble the pinion assembly.

2. Install the stopper ring to the pinion assembly.

3. Apply the grease to the gear.

4. Install the pinion assembly to the armature.

5. Install the bearing retainer to the gear case.

4. Yoke installation

1. Install the armature to the yoke.

2. Install the brush to the brush holder.

Note

  • Install it by raising the spring end of the brush spring.

Caution

  • Take care not to damage the commutator surface.

3. Install the brush holder to the yoke.

Note

  • Install it while aligning the peripheries of the yoke and the brush holder.

4. Install the yoke to the gear case.

Note

  • Install it together with the armature and the brush holder.

5. Rear cover installation

1. Install the rear cover to the motor assembly.

Tightening torque： 8 N・m { 0.8 kgf・m / 72 lb・in }

2. Connect the lead wire to the magnetic switch.

Tightening torque： 9 N・m { 0.9 kgf・m / 76 lb・in }

6. Pinion assembly measurement

1. Measure play using a vernier caliper.

Note

  • Measure the play of the pinion shaft thrust.
  • The play in the pinion shaft thrust is equal to the clearance between the pinion shaft end and the pinion stopper.

Clearance： 0.1 to 0.2 mm { 0.004 to 0.008 in }

7. Magnetic switch inspection

Caution

  • The following tests must be performed with the starter motor fully assembled.
  • Do not connect the lead wire of the yoke to the M-terminal.
  • To prevent coil burning, complete each test within 3 to 5 seconds as far as possible.

Note

  • Temporarily connect the solenoid switch between the clutch and the housing and perform the following test.

1. Inspect the magnetic switch.

Note

  • Pull-in test
  • Connect the battery negative terminal with the solenoid switch body and the M-terminal.
  • When the current flows to the S-terminal from the battery positive terminal, the pinion should flutter.

2. Inspect the magnetic switch.

Note

  • Hold-in maintenance test
  • Disconnect the lead at the M-terminal.
  • The pinion should continue to flutter.

3. Inspect the magnetic switch.

Note

  • Return test
  • Disconnect the battery positive lead at the S-terminal.
  • The pinion should return to its original position.
