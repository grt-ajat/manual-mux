Disc brake pad installation (All models)

1. Disc brake pad installation

Note

  • Both the front disc brake pad and rear disc brake pad can be installed using the same procedure.

1. Install the clip to the brake support.

Note

  • When replacing the pad, dispose of the used clip and use a new one.

Caution

  • Do not mix up the upper and lower sides of the clip.

  1. Lower side clip
  2. Upper side clip
  3. Clamp

2. Install the disc brake pad to the brake support.

Note

  • Apply grease to the back plate of the front disc brake pad.

Caution

  • Do not apply to the rear disc break pad.

  1. 10.3 mm (0.41 in)
  2. 5.0 mm (0.20 in)
  3. Section to which grease is applied
  4. 5.0 mm (0.20 in)
  5. 5.0 mm (0.20 in)

Note

  • Install the pad assembly attached with the shim to the brake support.
  • Align the inner pad indicator with the lower side.
  • After assembling the pad, if grease is found protruding again, wipe it off.
  • Front disc brake pad

  1. Stainless steel shim
  2. Outer pad
  3. Inner pad
  4. Indicator
  5. Rubber coated shim

Note

  • Rear disc brake pad

  1. Seven layered shim
  2. Outer pad
  3. Inner pad
  4. Indicator

Note

  • Lower the brake caliper to the original position.

Caution

  • Do not damage the brake hose by twisting or pulling it.

Caution

  • Confirm that the dust boot does not protrude from the piston end, and then lower the caliper.
  • Take care to prevent the dust boot from becoming caught between the piston and pad.

3. Install the lock bolt to the brake caliper assembly.

Tightening torque： 32 to 42 N・m { 3.3 to 4.3 kgf・m / 24 to 31 lb・ft }

2. Disc wheel installation

1. Install the disc wheel to vehicle.

Note

  • Install the wheel nuts in the numbered order.

Tightening torque: 120 N⋅m {12.0 kgf⋅m/87 lb⋅ft}
