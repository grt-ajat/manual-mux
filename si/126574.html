Cruise control system check (4JK1)

1. Cruise control system check description of function

The cruise control system consists of the ECM, the cruise main switch, the cruise control switch, and the cancel switch. The cruise control maintains the vehicle speed at a speed set by a driver. If the cruise main switch is turned ON, a signal is transmitted to the ECM, and the cruise main indicator light on the instrument panel cluster illuminates. If the cruise control set switch is operated, the switch signal is sent to the ECM and the vehicle speed is set. If the cruise control resume switch or the cruise control set switch is operated while the cruise control system is running, the vehicle speed will be increased or decreased. If the cruise cancel switch is turned ON, the switch signal is sent to the ECM and the cruise control system is deactivated.

Note

  • Conditions for running the cruise control

The vehicle speed is approximately 40 km/h (24 mph) or more.
The engine speed is 4500 rpm or less.
The cruise main switch is ON.

Function of SET
If the cruise control set switch is operated and released while the conditions for running the cruise control are met, the ECM memorizes and maintains the vehicle speed at that time.

Function of COAST
If the cruise control set switch is operated while the cruise control system is running, the vehicle speed will be decreased. Then, if the cruise control set switch is released, the vehicle will maintain the vehicle speed at that time.

Function of RESUME
If the cruise control resume switch is operated while the cruise control system is running, the ECM memorizes the vehicle speed, and the vehicle speed will be returned to the vehicle speed memorized by the ECM.

Function of ACCEL
If the cruise control resume switch is operated while the cruise control system is running, the vehicle speed will be increased. Then, if the cruise control resume switch is released, the vehicle will maintain the vehicle speed at that time.

Function of temporary acceleration
If the accelerator pedal is depressed while the cruise control system is running, the vehicle speed will be increased.

Function of temporary cancellation
The cruise control is canceled temporarily if any of the following conditions are met.
The cruise cancel switch is ON.
The brake pedal is depressed.
The clutch pedal is depressed (for MT).
The selector lever position is not D, 3, 2, or L (for AT).
The actual vehicle speed becomes less than 35 km/h (22 mph).
The actual vehicle speed exceeds the set speed by more than or equal to 40 km/h (24 mph), or exceeds the set speed by more than or equal to 10 km/h (6 mph) for 3 minutes or longer.
The actual vehicle speed is less than the set speed by more than or equal to 70 km/h (42 mph), or is less than the set speed by more than or equal to 10 km/h (6 mph) for 3 minutes.
If the cruise control resume switch is operated while the conditions for running the cruise control are met, the vehicle speed is returned to the vehicle speed memorized by the ECM.

Function of complete cancellation
The cruise control is canceled completely if any of the following conditions are met.
The cruise main switch is OFF.
The ignition switch is OFF.
The vehicle is temporarily stopped.
A related DTC that inhibits the cruise control system is set.

2. Cruise control system check

1. Diagnostic system check

Refer to "202.Description Engine 15C.Functional Inspection(4JK1) diagnostic system check-engine controls".

2. Operation verification 1

Drive the vehicle at a speed of 50 km/h (30 mph) on a flat road.

Press and release the cruise control main switch.

Verify that the vehicle speed is maintained when the cruise control set switch is operated and released.

Verify that the vehicle speed decreases in increments of 1 km/h (0.6 mph) when the cruise control set switch is operated again.

If the vehicle speed is maintained and decreased according to the switch operations, go to Operation verification 2.

If the vehicle speed is not maintained and decreased according to the switch operations, go to DTC verification.

3. Operation verification 2

Operate the cruise control resume switch 10 times.

Verify that the vehicle speed increases in increments of 1 km/h (0.6 mph) when the switch is operated.

If the vehicle speed is increased, go to Operation verification 3.

If the vehicle speed is not increased, go to Inspection of cruise control switch.

4. Operation verification 3

Verify that the cruise control main light or the cruise control set light illuminates during the cruise control.

If the light illuminates, go to Operation verification 4.

If the light does not illuminate, go to Instrument panel cluster setting verification.

5. Operation verification 4

Apply the cruise control cancel switch.

Go to Operation verification 5 if the cruise control is canceled.

If the cruise control is not canceled, go to Cruise control cancel switch circuit diagnosis.

6. Operation verification 5

Re-enter the cruise control.

Depress and release the brake pedal.

The system is normal if the cruise control is canceled.

Go to DTC P0571 diagnosis if the cruise control is not canceled.

7. DTC verification

Park the vehicle.

Install a scan tool.

Observe the DTC information with a scan tool.

When a DTC is set, if DTCs inhibiting the cruise control are set, go to the applicable DTC diagnosis.

When a DTC is set, if DTCs inhibiting the cruise control are not set, go to Cruise control main switch circuit diagnosis 1.

8. Cruise control main switch circuit diagnosis 1

Observe the Cruise Main Switch display on the scan tool while operating the cruise control main switch.

Note

  • The scan tool displays ON when the switch is pressed, and it displays OFF when the switch is released.

If the Cruise Main Switch display switches properly, go to Inspection of cruise control switch.

If the Cruise Main Switch display does not switch properly, go to Cruise control main switch circuit diagnosis 2.

9. Cruise control main switch circuit diagnosis 2

Turn OFF the ignition switch.

Disconnect the cruise control switch harness connector.

Turn ON the ignition switch.

Connect a test lamp between the cruise control main switch circuits.

If the test lamp illuminates, go to Cruise control main switch circuit diagnosis 3.

If the test lamp does not illuminate, repair the open circuit or high resistance in the cruise control main switch power supply circuit between the ENGINE IG 10 A fuse and the cruise control switch. Go to Repair verification.

10. Cruise control main switch circuit diagnosis 3

Observe the Cruise Main Switch display on the scan tool while connecting a fused jumper wire between the cruise control main switch circuits of the cruise control switch harness connector.

Note

  • The scan tool displays ON when the fused jumper wire is connected, and it displays OFF when it is disconnected.

If the Cruise Main Switch display switches properly, go to Inspection of cruise control switch harness connector.

If the Cruise Main Switch display does not switch properly, go to Inspection for open circuit in cruise control main switch signal circuit.

11. Inspection for open circuit in cruise control main switch signal circuit

Inspect the cruise control main switch signal circuit for an open circuit or high resistance, and repair as necessary. Go to Repair verification.

If the circuit is normal, go to Inspection of ECM harness connector.

12. Inspection of cruise control switch

Turn OFF the ignition switch.

Disconnect the cruise control switch harness connector.

Inspect the cruise control switch, and repair as necessary. Go to Repair verification.

If the switch is normal, go to Inspection for open circuit or short circuit in cruise control switch circuit.

13. Cruise control cancel switch circuit diagnosis

Turn OFF the ignition switch.

Disconnect the cruise control switch harness connector.

Turn ON the ignition switch.

Observe the Cruise Cancel Switch display on the scan tool.

If OFF is displayed, replace the cruise control switch. Go to Repair verification.

If ON is displayed, go to Inspection for open circuit or short circuit in cruise control switch circuit.

14. Inspection for open circuit or short circuit in cruise control switch circuit

Inspect the cruise control switch signal circuit for an open circuit, high resistance, a short to the power supply circuit, a short to ground, or a short to the low reference circuit, and repair as necessary. Go to Repair verification.

Inspect the cruise control switch low reference circuit for an open circuit or high resistance, and repair as necessary. Go to Repair verification.

Note

  • The cruise control switch shares the low reference circuit with other sensors. A fault condition in the low reference circuit may set DTCs on sensors that share this circuit.

If the circuit is normal, go to Inspection of ECM harness connector.

15. Inspection of cruise control switch harness connector

Inspect the connection status of the cruise control switch harness connector, and repair as necessary. Go to Repair verification.

If the cruise control switch harness connector is normal, replace the cruise control switch. Go to Repair verification.

16. Inspection of ECM harness connector

Inspect the connection status of the ECM harness connector, and repair as necessary. Go to Repair verification.

If the connection status is normal, go to ECM replacement.

17. ECM replacement

Replace the ECM. Go to Repair verification.

Refer to "1.Engine 1J.Electrical(4JK1) ECM removal".

Refer to "1.Engine 1J.Electrical(4JK1) ECM installation".

Refer to "1.Engine 1J.Electrical(4JK1) ECM setting".

Refer to "1.Engine 1J.Electrical(4JK1) ECM writing".

18. Instrument panel cluster setting verification

Display the Module Information with a scan tool.

Verify that the Cruise Control Indicator Light setting is Available.

If Not Available is displayed, go to Instrument panel cluster writing.

If Available is displayed, go to Instrument panel cluster replacement.

19. Instrument panel cluster writing

Write the vehicle information into the instrument panel cluster. Go to Repair verification.

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) instrument panel cluster setting".

20. Instrument panel cluster replacement

Replace the instrument panel cluster. After replacing the instrument panel cluster, perform the instrument panel cluster setting. Go to Repair verification.

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) instrument panel cluster removal".

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) instrument panel cluster installation".

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) instrument panel cluster setting".

21. Vehicle repair check

Connect all of the disconnected harness connectors.

Turn OFF the ignition switch for 30 seconds.

Go to Operation verification 1.
