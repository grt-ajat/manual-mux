Check 4WD warning light inoperative (All models)

1. Check 4WD warning light inoperative description of function

The check 4WD warning light illuminates when the ignition switch is turned ON, and then it goes off after 3 seconds if a DTC has not been detected. The check 4WD warning light illuminates when the TCCM sends a light illumination request to the instrument panel cluster via CAN communication.

Note

  • Function of the check 4WD warning light

The warning light is installed inside the instrument panel cluster and illuminates when an error occurs to notify the driver that an inspection and repair must be performed as soon as possible.
If the check 4WD warning light remains ON responding to an instruction from the TCCM, a DTC has been detected.

Note

  • Lighting condition of the check 4WD warning light

The warning light illuminates for 3 seconds after the ignition switch is turned ON.
When the TCCM detects a malfunction, the warning light remains ON.

2. Check 4WD warning light inoperative

1. Diagnostic system check

Refer to "204.Description Driveline, Axle 17C.Functional Inspection(All models) diagnostic system check-transfer case controls".

2. Confirmation of instrument panel cluster setting

Display the Module Information with a scan tool.

Confirm that the 4WD Indicator Light setting is Available.

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) instrument panel cluster setting".

If Not Available is displayed, go to Instrument panel cluster writing.

If Available is displayed, go to Replacement of instrument panel cluster.

3. Instrument panel cluster writing

Write the vehicle information into the instrument panel cluster. Go to Repair verification.

Refer to "3. Instrument panel cluster setting" of "Instrument panel cluster setting".

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) instrument panel cluster setting".

4. Replacement of instrument panel cluster

Replace the instrument panel cluster. After replacing the instrument panel cluster, conduct the setting of the instrument panel cluster. Go to Repair verification.

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) instrument panel cluster removal".

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) instrument panel cluster installation".

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) instrument panel cluster setting".

5. Repair verification

Turn OFF the ignition switch for at least 30 seconds.

Turn ON the ignition switch.

Check the lighting of the check 4WD warning light.
