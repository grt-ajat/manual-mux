Parking brake cable inspection (All models)

1. Brake shoe assembly inspection

1. Remove the brake drum from the brake assembly.

Note

  • Check the brake shoe assembly for worn.
  • If the lining thickness is below the limit value, replace the brake shoe assembly.

Limit： 1.0 mm { 0.039 in }

2. Rear rotor inspection

1. Inspect the brake rotor.

Note

  • Measure the thickness of the brake rotor.

Standard： 18.0 mm { 0.709 in }

Limit： 16.6 mm { 0.654 in }

Note

  • Measure the runout of the brake rotor.

Standard： 0.075 mm { 0.00295 in }

Note

  • Correctly adjust the wheel bearing before inspection.
  • Using a dial gauge, measure runout at a location 10 mm {0.04 in} from the outer edge of the rotor surface that makes contact with the disc brake pad.

Note

  • Measure the degree of parallelization of the brake rotor.

Limit： 0.023 mm { 0.00091 in }

Note

  • The contact surface is within 0.023 mm {0.0009 in} of the position located 10 mm {0.04 in} inward from the rotor outer end.
  • Measure at 8 or more locations in the circumferential direction.

Note

  • Measure the inner diameter of the parking brake drum.

Standard： 210.0 mm { 8.268 in }

Limit： 211.5 mm { 8.327 in }

Note

  • Measure the drum runout.

Standard： 0.08 mm { 0.0031 in }

Limit： 0.15 mm { 0.0059 in }
