DTC P0219 (Flash Code 543) Engine Overspeed Condition (4JK1)

1. DTC P0219 diagnostics

Note

  • If there is electromagnetic interference in the CKP sensor circuits, this DTC may set.
  • This DTC setting may occur in the CMP sensor failure.

Caution

  • This DTC is caused by a condition of engine overrun due to a driver's mistake, for example, downshifting of a manual transmission on a steep downhill slope. Engine overrun may damage internal engine components.

1. Diagnostic system check

Refer to "202.Description Engine 15C.Functional Inspection(4JK1) diagnostic system check-engine controls".

2. Prioritized DTC

If DTCs P0335 or P0336 are set at the same time, go to the applicable DTC diagnosis.

3. Engine speed verification

Start the engine.

While raising the engine speed as necessary, observe the Engine Speed display on the scan tool.

Standard： 4800 r/min

If the specified value is exceeded at least once, go to Inspection of CKP sensor.

If the reading is less than the specified value, ask the driver if overrun is caused by gear slip-out, shift error, down-slope driving, etc. If engine overrun has been experienced, the engine must be inspected and repaired as necessary. Go to Repair verification.

4. Inspection of CKP sensor

Inspect the CKP sensor and the sensor rotor for the following conditions, and repair as necessary. Go to Repair verification.

Sensor damage
Loose or incorrectly installed sensor
Excessive gap
Foreign material passing between sensor and sensor rotor
Sensor rotor damage
Loose or incorrectly installed sensor rotor

If it is normal, replace the CKP sensor. Go to Repair verification.

Refer to "1.Engine 1B.Mechanical(4JK1) CKP sensor removal".

Refer to "1.Engine 1B.Mechanical(4JK1) CKP sensor installation".

2. DTC P0219 repair verification

Clear the DTC with a scan tool.

Turn OFF the ignition switch for at least 30 seconds.

Start the engine.

While observing the Engine Speed display on a scan tool, depress the accelerator pedal to W.O.T. for several times.

Standard： 4800 r/min

If the specified value is exceeded at least once, go to Inspection of CKP sensor.

If the reading is less than the specified value, observe the DTC information with a scan tool.
