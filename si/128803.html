Front disc brake inspection (All models)

1. Front disc brake inspection

1. Inspect the brake assembly.

Note

  • If any worn, damage, or other abnormalities are found during the inspection, make repairs or replace the parts.
  • Visually inspect the following parts for worn, bent, degradation, corrosion, abrasion, scratches, or other abnormalities.
  • Rotor
  • Caliper body
  • Cylinder bore
  • Piston
  • Brake support
  • Lock bolt
  • Guide pin

2. Brake rotor inspection

1. Inspect the brake rotor.

Note

  • Measure the thickness of the brake rotor.

Standard： 27.0 mm { 1.063 in }

Limit： 25.0 mm { 0.984 in }

Note

  • Measure the brake rotor fluctuation.

Limit： 0.075 mm { 0.00295 in }

Note

  • Adjust the wheel bearing properly before the inspection.
  • Measure the amount of fluctuation with a dial gauge at a position 10 mm {0.04 in} away from the rotor outside end on the contact surface of the disc brake pad.

Note

  • Measure the parallelism of the brake rotor.

Limit： 0.023 mm { 0.00091 in }

Note

  • The contact surface is within 0.023 mm {0.0009 in} of the position located 10 mm {0.04 in} inward from the rotor outer end.
  • Perform measurement at 8 locations or more in a circumferential direction.

3. Disc brake pad inspection

1. Inspect the disc brake pad.

Note

  • Measure the thickness of the disc brake pad.

Standard： 10.0 mm { 0.394 in } Front disc brake

Limit： 1.5 mm { 0.059 in } Front disc brake

Note

  • When the pad wear indicator makes a scratching sound or is worn to within 1.8 mm {0.071 in} of the shoe table, replace the pad.

Caution

  • When replacing, replace all 4 pads (inner and outer pads of both the left and right wheels).

4. Guide pin inspection

1. Inspect the guide pin.

2. Inspect the lock pin.

Note

  • Inspect it for sticking, worn, bending, degradation, corrosion, abrasion, scratches, or other abnormalities.
  • If there are abnormalities, make repairs or replace the parts.

  1. Support bracket
  2. Slide pin boot
  3. Slide pin (guide pin)
  4. Lock bolt
  5. Pad clip
  6. Slide pin (lock pin)

3. Inspect the boot.

Note

  • Replace the boot every time the caliper is overhauled.

4. Inspect the dust seal.

Note

  • Replace the dust seal and ring seal every time the caliper is overhauled.

  1. Piston
  2. Caliper body
  3. Dust boot
  4. Ring seal
