Diagnostic system check-SRS controls (All models)

1. Diagnostic system check-SRS controls description of function

The Diagnostic system check - SRS controls is an organized approach to identifying a problem created by an SRS control system malfunction. Using it correctly will reduce diagnostic time and prevent the replacement of good parts.

2. Diagnostic system check-SRS controls

Note

  • Check for a related Service Bulletin before starting the diagnostic.
  • Do not clear the DTCs unless it is instructed in a diagnostic step.
  • The battery should be fully charged.
  • The battery cable should be clean and securely connected.
  • The SRS control unit harness connector should be clean and securely connected.
  • The SRS control unit terminal should be clean and matched correctly.

1. Inspection of scan tool power up

Install a scan tool.

Verify whether the scan tool powers up.

If the scan tool does not power up, refer to Scan tool does not power up and repair as necessary.

2. Inspection of scan tool communication

Turn ON the ignition switch.

Verify the vehicle specifications and the status of communication with the following control units with the scan tool.

ECM

TCM (if so equipped)

TCCM (if so equipped)

EHCU (if so equipped)

ICU (if so equipped)

SRS control unit

BCM (if so equipped)

Instrument panel cluster

If communication can be established with all the control units, go to Inspection of SRS airbag warning light.

If there are any control units with which communication cannot be established, refer to Scan tool does not communicate with CAN device in the Description Engine section, and repair as necessary.

3. Inspection of SRS airbag warning light

Turn OFF the ignition switch for at least 15 seconds.

Turn ON the ignition switch.

Check the SRS airbag warning light.

Note

  • If the SRS function is normal, the SRS airbag warning light flashes 7 times before turning OFF.

If the SRS airbag warning light remains OFF without flashing 7 times, refer to Inspection of SRS airbag warning light circuit system, and repair as necessary.

If the SRS airbag warning light remains ON without flashing 7 times, refer to Inspection of SRS airbag warning light circuit system, and repair as necessary.

If the SRS airbag warning light remains ON after flashing 7 times, observe the DTC information with a scan tool.

Note

  • If DTCs B0052, B0053, B0055, B0056, B1005, B1008, B1009, or B101D are set, inspect the applicable DTCs first.
  • Detection of a DTC beginning with U indicates that there is no communication with other ECUs.
  • When multiple DTCs are set, diagnose a DTC starting with the one that has a smaller number unless otherwise instructed in the diagnostic steps.
