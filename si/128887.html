DTC C0050[18] (Flash Code 37) RR Abnormal wheel speed variation B (ABS/ESC)

1. DTC C0050[18] diagnostics

1. Diagnostic system check

Refer to "205.Description Brakes 18C.Functional Inspection(All models) diagnostic system check- ABS/EBD/TCS/ESC controls".

2. Prioritized DTC

If DTC C0050[06] is set at the same time, go to DTC C0050[06] diagnosis.

3. Inspection of harness connector

Inspect the connection status of the rear right wheel speed sensor harness connector, and repair as necessary. Go to Repair verification.

Inspect the connection status of the rear right wheel speed sensor circuit at the EHCU harness connector, and repair as necessary. Go to Repair verification.

If the connection status is normal, go to Inspection of rear right wheel speed sensor circuit.

4. Inspection of rear right wheel speed sensor circuit

Inspect for high resistance in the rear right wheel speed sensor circuit, and repair as necessary. Go to Repair verification.

If the circuit is normal, go to Inspection of rear right wheel speed sensor and sensor rotor.

5. Inspection of rear right wheel speed sensor and sensor rotor

Inspect the rear right wheel speed sensor and the sensor rotor for the following conditions, and repair as necessary. Go to Repair verification.

Damage.
Incorrect installation.
The gap between the wheel speed sensor and the sensor rotor is excessive.
There is electromagnetic interference in the wheel speed sensor circuit.
Foreign material passes through the gap between the wheel speed sensor and the sensor rotor.

If it is normal, replace the rear right wheel speed sensor. Go to Repair verification.

Refer to "4.Brakes 4D.ABS(All models) rear wheel speed sensor removal".

Refer to "4.Brakes 4D.ABS(All models) rear wheel speed sensor installation".

2. DTC C0050[18] repair verification

Clear the DTC with a scan tool.

Turn OFF the ignition switch.

Start the engine and drive the vehicle at a speed above 25 km/h (16 mph).

Observe the DTC information with a scan tool.
