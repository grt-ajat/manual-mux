Condenser assembly installation (All models)

1. Condenser assembly installation

1. Install the condenser assembly to vehicle.

Note

  • When installing a new condenser assembly, add 30 cm3 {0.85 lmp fl oz} of compressor oil.

  1. Condenser upper bracket
  2. Condenser assembly

2. A/C hose and A/C pipe installation

1. Connect the A/C hose and A/C pipe to the condenser assembly.

Caution

  • Do not reuse the O-ring, and a new one should be used.

Note

  • Make sure to apply a layer of compressor oil to the replaced O-ring before connecting the A/C pipe.
  • Tighten the A/C hose and 1 A/C pipe mounting bolt at the specified torque.

Tightening torque： 15 N・m { 1.5 kgf・m / 11 lb・ft }

Note

  • Install the A/C pipe mounting clamp.

  1. Condenser assembly
  2. High pressure hose
  3. High pressure pipe
  4. Bolt

3. Radiator support assembly installation

1. Install the radiator support assembly to vehicle.

Note

  • Install the engine room harness with the 5 clips and tighten the 8 mounting bolts.

  1. Condenser upper bracket
  2. Condenser assembly
  3. Radiator support assembly

4. Condenser upper bracket installation

1. Install the condenser upper bracket to vehicle.

Note

  • Tighten 1 bolt each for the left and right.

  1. Condenser upper bracket
  2. Condenser assembly
  3. Radiator support assembly

5. Radiator upper mounting bracket installation

1. Install the radiator upper mounting bracket to vehicle.

Note

  • Tighten 1 bolt each for the left and right.

  1. Bolt
  2. Radiator upper mounting bracket
  3. Radiator support assembly

6. Engine hood lock assembly installation

1. Temporarily tighten the engine hood lock assembly to the body.

Note

  • Connect the engine food lock switch connector.
  • Install the engine hood lock assembly to the body, and temporarily tighten the bolt.

2. Close the engine hood assembly.

Note

  • Position the hood so that the level difference between the hood and the fender becomes the specified value.
  • After confirming dimensions, tighten the fixing bolt of the engine hood assembly.
  • After tightening the fixing bolt, check the level difference again. If the value is out of the specified value, adjust the level difference by turning the hood rest.

  1. Engine hood assembly
  2. Front fender
  3. Hood rest
  4. Engine hood lock assembly
  5. Specified dimension: (-0.5) - (-2.5) mm {(-0.02) - (-0.10) in}

3. Securely tighten the engine hood lock assembly to the body.

Note

  • Tighten the fixing bolt to the specified torque.

Tightening torque： 10 N・m { 1.0 kgf・m / 89 lb・in }

  2. Engine hood lock cover

10. Intercooler installation

1. Install the intercooler to the frame.

Tightening torque： 20 N・m { 2.0 kgf・m / 15 lb・ft }

2. Connect the intake air hose to the intercooler.

Note

  • Connect it at both the turbocharger and intake throttle sides.

Tightening torque： 5 N・m { 0.5 kgf・m / 44 lb・in }

  1. Intake hose (Turbocharger side)
  2. Intake hose (Intake throttle side)

1. Install the front end engine hood stay to the frame.
