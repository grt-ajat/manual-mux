Diagnostic system check-transfer case controls (All models)

1. Diagnostic system check-transfer case controls description of function

The diagnostic system check-transfer case controls are a sequenced method for checking problems caused by the functional failures in the transfer case control system, and are also the starting point for all diagnoses for drivability complaints. By correctly using these diagnostic steps, it is possible to reduce the time for diagnosis and prevent replacing parts that are not defective.

2. Diagnostic system check-transfer case controls

Note

  • If there are no complaints regarding drivability, do not perform these diagnostic steps unless otherwise instructed to do so.
  • Check the Service Bulletin before starting the diagnosis.
  • Do not clear the DTCs unless it is instructed in a diagnostic step.
  • The battery should be fully charged.
  • The battery cable should be clean and securely connected.
  • The ground of the TCCM should be clean and securely connected in the correct position.
  • The TCCM harness connector should be clean and securely connected.
  • The TCCM terminal should be clean and matched correctly.

1. Inspection of scan tool power up

Install a scan tool.

Verify whether the scan tool powers up.

If the scan tool does not power up, refer to Scan tool does not power up, and repair as necessary.

2. Inspection of scan tool communication

Turn ON the ignition switch.

Verify the vehicle specifications and the status of communication with the following control units with the scan tool.

ECM
TCM (if so equipped)
TCCM
EHCU (if so equipped)
ICU (if so equipped)
SRS control unit (if so equipped)
BCM (if so equipped)
Instrument panel cluster

When communication can be made with all the control units, go to Inspection of check 4WD warning light.

If there are any control units with which communication has not been established, refer to Scan tool does not communicate with CAN device in the Description Engine section, and repair as necessary.

3. Inspection of check 4WD warning light

Turn OFF the ignition switch.

Turn ON the ignition switch.

Check the check 4WD warning light.

Note

  • The check 4WD warning light illuminates when the ignition switch is turned ON, and goes off after approx. 3 seconds.

If the check 4WD warning light does not illuminate, refer to Check 4WD warning light inoperative and repair.

If the check 4WD warning light does not turn off, refer to Check 4WD warning light always on and repair.

4. Inspection of 4WD indicator light and 4WD low indicator light

Drive the vehicle in 4H and 4L modes respectively.

If the mode does not switch properly, refer to 4WD switch system check, and inspect.

Verify the 4WD indicator light and the 4WD low indicator light.

Note

  • The 4WD indicator light illuminates when the 4WD switch is shifted to the 4H or 4L position.
  • When shifting to the 4L position, the 4WD low indicator light illuminates.

If the 4WD indicator light and the 4WD low indicator light do not illuminate, refer to 4WD, 4WD Low indicator light inoperative and repair it.

If the 4WD indicator light and the 4WD low indicator light do not turn OFF, refer to 4WD, 4WD low indicator light always on and repair it.

5. Checking DTCs

Observe the TCCM DTC information with a scan tool.

If a DTC beginning with U or a DTC related to communication with other control modules has been detected, diagnose the applicable DTCs.

If TCCM DTCs C0867 or C0868 are set, diagnose the applicable DTCs.

If a DTC other than the above DTC has been detected, go to the applicable DTC diagnosis.

Note

  • When multiple DTCs are set, start diagnosing in numerical order of the DTCs unless otherwise instructed in the diagnostic step.

6. Confirmation 1 of symptom

Verify the customer's concern is related with the following systems.

Engine
Transmission

If it is applicable, refer to Diagnostic system check for the relevant system and inspect it.

7. Confirmation 2 of symptom

Check if any of the following conditions apply.

Abnormal noise and vibration from propeller shaft.
Power not being transmitted to the wheels.
Abnormal noise and vibration from axle.
Intermittent conditions of transfer case control system.
Oil leakage from axle.

If a problem is found, repair the relevant symptoms.
If it is not applicable to any symptom, refer to Intermittent conditions of transfer case control system and inspect it.
