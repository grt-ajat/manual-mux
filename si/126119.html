Crankshaft inspection (4JK1)

1. Thrust bearing setting

1. Measure the clearance using a feeler gauge.

Note

  • Measure the crankshaft thrust clearance at the center part of the crankshaft journal.
  • Replace the thrust bearing if the measured value exceeds the specified value.

Standard： 0.040 to 0.201 mm { 0.0016 to 0.0079 in }

Caution

  • Measure the thrust clearance prior to disassembly.

2. Crankshaft bearing measurement

1. Remove the bearing cap from the cylinder block.

2. Remove the lower crankshaft bearing from the bearing cap.

Note

  • The removed crankshaft bearing should be lined in the order of the number.

3. Remove the crankshaft from the cylinder block.

4. Remove the upper crankshaft bearing from the cylinder block.

Note

  • The removed crankshaft bearing should be lined in the order of the number.

5. Inspect the crankshaft bearing.

Note

  • Clean the crankshaft journal and upper and lower bearings.
  • Inspect the bearing for damage or excessive wear.
  • If damage or excessive wear is found, replace the bearings as a set.

6. Install the upper crankshaft bearing to the cylinder block.

7. Install the crankshaft to the cylinder block.

Note

  • Install the crankshaft to be level.

8. Put the PLASTIGAUGE on the crankshaft.

Note

  • As shown in the diagram, place the PLASTIGAUGE on the crankshaft journal.

9. Install the lower crankshaft bearing to the bearing cap.

10. Apply the engine oil to the bolt.

Note

  • Apply engine oil to the threaded portion of the bolt and the seat surface.

11. Install the bearing cap to the cylinder block.

Note

  • Tighten the bolts to the specified torque in the order shown in the diagram.

Tightening torque： 166 N・m { 16.9 kgf・m / 122.4 lb・ft }

Caution

  • After tightening the bolts, do not turn the crankshaft.

12. Remove the bearing cap bolt from the cylinder block.

Note

  • Remove it gently.

13. Measure the oil clearance using the PLASTIGAUGE.

Note

  • Measure the maximum width portion of the PLASTIGAUGEs crushed by the tightening of the bearing caps.
  • Determine the oil clearance of the journal.

Standard： 0.030 to 0.054 mm { 0.0012 to 0.0021 in }

Note

  • If the oil clearance of the journal exceeds the specified value, replace all crankshaft bearings or the crankshaft.
  • Remove the PLASTIGAUGE from the bearing and the crankshaft.

3. Crankshaft inspection

1. Inspect the crankshaft.

Note

  • Check the crankshaft journal and crank pin surface for wear and damage.
  • Check the contact surface of the oil seal for excessive wear and damage.
  • Check the oil port for clogging.

4. Crankshaft measurement

1. Put the crankshaft on the V-block.

Note

  • Carefully set the crankshaft.

2. Measure fluctuation using a dial gauge.

Note

  • Slowly rotate the crankshaft to measure the runout.
  • Replace the crankshaft if the crankshaft fluctuation exceeds the specified value.

Standard： 0.05 mm or less { 0.0020 in or less }

3. Measure outer diameter using the micrometer.

Note

  • Measure the journal and the pin outer diameters to calculate the difference of the maximum and minimum.
  • Measure in 4 sections on the journal and on the pin.

Standard： 69.917 to 69.932 mm { 2.7526 to 2.7532 in } Journal section

Standard： 52.915 to 52.930 mm { 2.0833 to 2.0839 in } Pin

Note

  • Inspection of the TUFFTRIDE layer of the crankshaft

Caution

  • TUFFTRIDE (soft nitriding treatment) is implemented to increase the strength of the crankshaft.
  • Therefore, the crankshaft surface cannot be polished.

4. Clean the crankshaft using organic solvent.

Note

  • Thoroughly clean the crankshaft to completely remove the oil from the test locations.

5. Prepare inspection solution.

Note

  • Prepare a solution of copper chloride ammonium of 5 to 10 %.

6. Apply inspection solution to the crankshaft.

Note

  • Use an oil drop to apply the solution to the test locations.
  • Maintain the inspection surface at a complete level so that the solution does not flow.

  1. Drip prohibited area
  2. Test liquid drop section
  3. Pin or journal sliding surface

Caution

  • Do not allow the solution to come into contact with the oil port.

Note

  • Wait 30 to 40 seconds.
  • If the color does not change after 30 or 40 seconds, it indicates a residual layer of TUFFTRIDE and the crankshaft can be used.
  • If the test surface turns copper-colored, replace the crankshaft.

Caution

  • Copper chloride ammonium solution is highly corrosive.
  • Therefore, it is important to immediately clean the test surface when the test is completed.
