Engine start/stop button installation (All models)

1. Engine start/stop button installation

1. Install the engine start/stop button to the instrument panel driver-side lower cover.

  1. Engine start/stop button
  2. Instrument panel driver-side lower cover

2. Instrument panel driver-side lower cover installation

1. Install the instrument panel driver-side lower cover to the instrument panel.

Note

  • Connect the switch connector.
  • Install the DLC.

  1. Instrument panel driver-side lower cover
  2. DLC

2. Install the fuel filler lid opener lever to the instrument panel driver-side lower cover.

Note

  • Install the 2 fixing screws.

  1. Fuel filler rid opener lever
  2. Fuel filler rid cable
  3. Screw

3. Install the hood lock control lever to the instrument panel driver-side lower cover.

Note

  • Install the 2 fixing screws.

  1. Hood lock control lever
  2. Control cable
  3. Screw

3. Battery ground cable connect

1. Connect the battery ground cable to the battery.
