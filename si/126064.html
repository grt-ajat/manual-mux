Starter motor inspection (4JJ1)

1. Armature measurement

1. Measure the commutator using a dial gauge.

Note

  • Measure the fluctuation, and replace the armature if the measured value exceeds the limit.

Standard： 0.05 mm { 0.0020 in }

Limit： 0.25 mm { 0.0098 in }

2. Inspect the commutator.

Note

  • Check the mica segment for excessive worn.

3. Measure the commutator using a vernier caliper.

Note

  • Measure the mica segment depth.

Standard： 0.5 to 0.8 mm { 0.020 to 0.031 in }

Limit： 0.2 mm { 0.008 in }

Note

  • If the mica segment depth is lower than the standard value but larger than the limit, the commutator can be re-ground.
  • If the mica segment depth is less than the limit, the armature must be replaced.

  2. Measurement point (undercut)

4. Measure the commutator using a vernier caliper.

Note

  • Measure the outer diameter.
  • If the measured outer diameter is less than the specified limit, the armature must be replaced.

Standard： 36.5 mm { 1.437 in }

Limit： 35.5 mm { 1.398 in }

5. Measure the armature using the circuit tester.

Note

  • Check whether the armature is grounded.
  • Hold one probe against the commutator segment.
  • Hold the other probe against the armature core.
  • If there is continuity, the armature is grounded, so replace the armature.

6. Measure the armature using the circuit tester.

Note

  • Check the armature for continuity.
  • Hold the probes against 2 locations of the commutator segment.
  • Hold the probes also against other areas.
  • There should be continuity between all segments of the commutator.
  • Otherwise, the armature must be replaced.

2. Yoke measurement

1. Measure the yoke using the circuit tester.

Note

  • Check the field winding ground.
  • Hold one probe against the field winding end or brush.
  • Hold the other probe against the bare surface of the yoke body.
  • If there is continuity, replace the yoke.

Note

  • Confirm the continuity of the field winding.
  • Hold one probe against the M-terminal lead wire.
  • Hold the other probe against the field winding brush.
  • If there is no continuity, replace the yoke.

3. Brush holder measurement

1. Measure the brush using a vernier caliper.

Note

  • Replace all the brushes if the measured brush length is less than the specified limit.

Standard： 15 mm { 0.59 in }

Limit： 12 mm { 0.47 in }

2. Measure the brush holder using the circuit tester.

Note

  • Check the insulation.
  • Hold one probe against the holder plate.
  • Hold the other probe against the plus brush holder.
  • There should be no continuity.

3. Inspect the brush.

Note

  • Inspect for excessive worn.
  • When the negative brushes have excessive worn, the brush holder assembly must be replaced.
  • When the positive brushes have excessive worn, the yoke must be replaced.

4. Pinion assembly inspection

1. Inspect the overrunning clutch.

Note

  • Inspect the gear teeth for excessive worn and damage.
  • Replace the overrunning clutch if necessary.

2. Turn the pinion.

Note

  • Rotate it clockwise.
  • It should turn smoothly.
  • Rotate the pinion in the opposite direction.
  • The pinion should lock.
