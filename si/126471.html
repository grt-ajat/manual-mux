White smoke (4JJ1)

1. White smoke description of symptom

White smoke is generated when the engine is under load, idling or hot, or when starting the engine in a cold state.

2. White smoke diagnostics

Note

  • Preliminary inspection

Check if any problem exists in the vehicle.
Ensure that the ground of the ECM is free of dirt and is fixed securely in the correct position.
Inspect the fuel quality.
Verify the injector ID code of each cylinder.
Check the scan tool data list.
Check for related Service Bulletins.

Note

  • Sensor inspection

Compare each display for Engine Coolant Temperature (ECT), Intake Air Temperature (IAT), and Fuel Temperature with the scan tool data list when the engine is stopped and cold. If the difference between the temperature values is 5 °C (9 °F) or more when the engine has cooled, inspect for high resistance in each circuit and for sensor failure.
{When the ignition switch is turned ON, the MAF sensor is heated, and as a result, the IAT shows a higher value than the normal intake air temperature.}
Observe the Fuel Rail Pressure (FRP) Sensor display on the scan tool. It is normal if the Fuel Rail Pressure (FRP) Sensor display shows 0.9 - 1.0 V when the engine is not started with the ignition switch turned ON after at least 1 minute from the stop of engine operation. If not, inspect for high resistance in each circuit and for any sensor failure.
Fully depress the accelerator pedal while idling in neutral, and verify the Fuel Rail Pressure (FRP) and Desired Fuel Rail Pressure (FRP) displays. It is normal if the Fuel Rail Pressure (FRP) display and the Desired Fuel Rail Pressure (FRP) display follow each other within a difference of 5 MPa {725 psi}.
Observe the accelerator pedal position. It is normal if the Accelerator Pedal Position (APP) display changes linearly from 0 to 100% according to the accelerator pedal operation.
Observe the Boost Pressure and Barometric Pressure (BARO) displays when the ignition switch is ON and the engine is OFF. It is normal if the difference between both data displays is within 7.0 kPa (1.0 psi).
The CKP sensor is securely installed, and the sensor rotor is not damaged.

Note

  • Fuel system inspection

If excessive smoke is present, inspect the stuck close injector. Inspect the fuel leaks into the combustion chamber.
Inspect for water intrusion in the fuel.
Remove the injector, and visually inspect it.
Perform the cylinder balance test with the scan tool.
Perform the Pre Injection Control with the scan tool.
Confirm the fuel compensation display of each cylinder with the scan tool while idling.

Note

  • Air intake system inspection

Inspect the air cleaner and air intake duct for clogging, holes and leakage.
Inspect for any condition that restricts the flow within the turbocharger inlet duct.
Inspect for stuck-close intake throttle valve.
Inspect for clogging or air leakage in the intake manifold.
Inspect the MAF sensor for dirt and damage.
Perform the swirl control solenoid control with the scan tool. Inspect the diaphragm valve operation when the ON/OFF command is executed on the diaphragm valve.
Inspect the turbocharger turbine wheel, shaft and compressor wheel for worn and damage.
Inspect the turbocharger oil seal for oil leakage.
Inspect the turbocharger oil return pipe for clogging.

Note

  • Engine mechanical inspection

Inspect the compression pressure of the engine.
Check for improper idle gear timing.
Inspect the valve clearance.
Inspect the valve stem for sticking.
Inspect for the valve stem oil seal failure.
Inspect the valve seat for worn.
Inspect the valve spring for damage and fatigue.
Inspect the camshaft for worn.
Inspect for compression leakage due to piston ring damage.
Inspect for gasket failure.
Inspect the piston for seizure.
Inspect for oil intrusion into the combustion chamber.

Note

  • Electrical system inspection

Inspect the glow plug control system operation.
