Front hub and brake rotor (2WD high ride suspension) (4WD) reassembly (All models)

1. Wheel pin installation

1. Install the wheel pin to the front hub.

Note

  • To protect the wheel pin end and the screw, place the hub on a wooden working table or piece of wood of about 150 x 150 mm {6 x 6 in}.
  • Use a hammer to tap the wheel pin straight securely to the end.

2. Front hub rotor assembly reassembly

1. Install the brake rotor to the front hub.

Note

  • Align the alignment marks that were made during disassembly to install.

Note

  • Use a protective pad to pinch the front hub rotor assembly with vise and tighten 6 bolts fixing the rotor and hub.

Tightening torque： 93 to 113 N・m { 9.5 to 11.5 kgf・m / 69 to 83 lb・ft }

3. Sensor rotor installation

1. Install the sensor rotor to the front hub rotor assembly using special tool.

Note

  • Use the sensor rotor installer and grip to install the sensor rotor in the hub.

Caution

  • Always use a new sensor rotor.


SST: 5-8840-2789-0 - sensor rotor installer


SST: 5-8840-0007-0 - grip

  1. Front hub rotor assembly
  2. Sensor rotor
  3. Push

2. Measure the sensor rotor using the gauge.

Note

  • Measure the gap.
  • Conduct measurement at 4 points of every 90° interval.
  • Confirm that the gap is within the specified range.

Standard： 7.0 to 7.4 mm { 0.276 to 0.291 in } Clearance

4. Inner bearing installation

1. Install the inner bearing to the front hub using special tool.

Note

  • Install the inner bearing to the front hub using an inner bearing installer and grip.


SST: 5-8840-2119-0 - installer: inner bearing


SST: 5-8840-0007-0 - grip

  1. Grip
  2. Inner bearing installer

5. Outer bearing installation

1. Install the outer bearing to the front hub using special tool.

Note

  • Install the outer bearing to the front hub using a bearing installer and grip.


SST: 5-8522-0054-0 - bearing installer


SST: 5-8840-0007-0 - grip

  1. Bearing installer
  2. Grip

6. Oil seal installation

1. Install the oil seal to the front hub using special tool.

Note

  • Install using an oil seal installer and grip.
  • Apply BESCO L-2 grease or equivalent to the lip of the oil seal.
  • Use a new oil seal.


SST: 5-8840-2854-0 - oil seal installer


SST: 5-8840-0007-0 - grip

  1. Grip
  2. Oil seal installer
