Piston and connecting rod installation (4JJ1)

1. Piston installation

1. Install the connecting rod bearing to the connecting rod.

2. Apply the engine oil to the connecting rod bearing.

Note

  • Apply to the sliding surface of the connecting rod bearing.

Caution

  • Do not apply engine oil to the rear surface of the bearing or the connecting rod bearing mounting surface.

3. Apply the engine oil to the piston ring.

Note

  • Sufficiently apply the engine oil to the piston ring and piston ring grooves.

4. Apply the engine oil to the piston.

Note

  • Sufficiently apply the engine oil to the piston side surface.

5. Apply the engine oil to the cylinder block.

Note

  • Apply the engine oil to the cylinder bore surface.

6. Turn the crankshaft.

Note

  • Turn the crankshaft in the forward direction and align to top dead center.

7. Install the piston to the cylinder block using special tool.

Note

  • Face the piston front mark towards the front.
  • Insert the piston into the cylinder block using a piston ring compressor.


SST: 5-8840-9018-0 - piston setting tool

Caution

  • When pushing the piston, be careful not to touch the oil jet against the connecting rod.
  • When pushing the piston, be careful not to damage the inside of the cylinder block.
  • If the crank pin is at bottom dead center when installing the piston and the connecting rod, the connecting rod may hit and damage the oil jet.

8. Install the connecting rod bearing to the connecting rod bearing cap.

9. Apply the engine oil to the connecting rod bearing.

Note

  • Apply to the sliding surface of the connecting rod bearing.

Caution

  • Do not apply engine oil to the rear surface of the bearing or the connecting rod bearing cap mounting surface.

10. Apply the engine oil to the bolt.

Note

  • Apply it to the threaded portion and the seat surface of the tightening bolts.

11. Temporarily tighten the connecting rod bearing cap to the connecting rod.

Note

  • Align the cap and the connecting rod number markings and install the cap.

12. Securely tighten the connecting rod bearing cap to the connecting rod.

Note

  • Tighten the bolts to the specified torque in the order shown in the diagram.

Tightening torque： 29.4 N・m { 3.0 kgf・m / 22 lb・ft }

13. Securely tighten the connecting rod bearing cap to the connecting rod.


SST: 5-8840-0266-0 - angle gauge

Note

  • Tighten the bolts to the specified angle in the order shown in the diagram.

Specified angle： 45 to 60 °

Note

  • Confirm that the crankshaft turns smoothly.

1. Select the cylinder head gasket.

Note

  • Select the grade for the cylinder head gasket by measuring the protrusion amount of the piston head.
  • Three types of gaskets with different thickness are provided.
  • Select the appropriate grade from three types of gaskets following the following steps.
  • Before measurement, remove carbon from the piston head and the cylinder block surface, and clean the gasket installation portion.

2. Measure the piston using a dial gauge.

Note

  • Measure the protrusion amount of the piston head.

Note

  • Refer to the diagram for the measurement position of the protrusion amount.
  • Make sure that the measurement point is as close to the cylinder block as possible.

3. Calculate average value from measured value.

Note

  • Calculate the average value for the protrusion amount of the piston head for each cylinder.
  • Select the gasket grade required by the aforementioned average value using the chart below.
