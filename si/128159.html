Clutch control installation (MUA)

1. Damper cylinder assembly installation

1. Install the damper cylinder assembly to the frame.

2. Clutch slave cylinder installation

1. Install the clutch slave cylinder to the transmission.

2. Connect the oil pipe to the clutch slave cylinder.

Note

  • Connect to the clutch slave cylinder and damper cylinder assembly.

Tightening torque： 34 N・m { 3.5 kgf・m / 25 lb・ft }

3. Clutch master cylinder installation

1. Install the clutch master cylinder to the frame.

2. Connect the oil pipe to the clutch master cylinder.

4. Clutch pedal installation

1. Apply the grease to the spring.

Note

  • Apply BESCO chassis grease or equivalent to the clutch pedal spring.

2. Install the clutch pedal to the frame.

3. Apply the grease to the clevis pin.

Note

  • Apply BESCO chassis grease or equivalent.

4. Install the clevis pin to the clutch pedal.

5. Install the pin to the clevis pin.

5. Clutch pedal inspection

1. Inspect play.

Note

  • Lightly press the clutch pedal with a hand to inspect that the play is within the standard value range.

Standard： 5.0 to 15.0 mm { 0.2 to 0.6 in }

  1. Play

6. Clutch pedal adjustment

1. Loosen the lock nut using a wrench.

2. Turn the clutch switch.

Note

  • Turn the clutch switch and adjust so that the clearance is within the standard value range.
  • After adjustment, tighten the lock nut.

Clearance： 0.5 to 1.5 mm { 0.020 to 0.059 in }

  1. Clutch switch
  2. Lock nut
  3. Clearance

7. Clutch fluid air bleed

1. Apply the parking brake.

Note

  • Pull the parking brake lever to a full extent.

2. Replenish the reservoir tank with the brake fluid.

Note

  • The reservoir tank of the clutch fluid is shared with the reservoir tank of the brake fluid.
  • Clean the pathway of the clutch master cylinder, clutch slave cylinder, and piping, etc. using an air blower.
  • Fill the brake fluid to MAX level, and wait for 1 minute or longer.

Caution

  • As the amount of clutch fluid is reduced through the air removal process, check frequently to make sure the reservoir tank is not emptied.

  1. Reservoir tank
  2. Clutch master cylinder

Note

  • As the quantity of the fluid is different between the clutch fluid chamber and brake fluid chamber depending on the partition inside the reservoir tank, check the quantity of clutch fluid in the clutch fluid chamber.

  1. MAX level
  2. MIN level
  3. Clutch fluid compartment

3. Install the plastic tube to the bleeder screw.

4. Install the plastic tube to the container.

Note

  • Insert the one edge of the plastic tube into the container filled with brake fluid.

5. Press the clutch pedal.

Note

  • Depress the clutch pedal 3 times and hold it down.
  • Depress the clutch pedal at the rate of 60 mm/sec, and return the pedal at the rate of 40 mm/sec.

6. Loosen the bleeder screw using a wrench.

Note

  • Loosen the bleeder screw 1/3 rotation to drain the brake fluid.

7. Tighten the bleeder screw using a wrench.

Note

  • Loose the bleeder screw, and the return the clutch pedal.
  • Repeat the process from depressing the clutch pedal to opening/closing the bleeder screw at intervals of 2 seconds.

Caution

  • Repeat 30 times or more until the air has been completely bled out.
