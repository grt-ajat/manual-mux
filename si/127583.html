Tachometer representational accuracy check inspection (All models)

1. Tachometer representational accuracy check

1. Connect a scan tool to the DLC.

2. Start the engine.

Note

  • Use a scan tool to display the engine rpm.
  • Compare the tachometer display and scan tool display.
Tester indicatorMeter indicator
Tolerance range
750 rpm700 - 870 rpm
2000 rpm1,915 - 2,085 rpm
3000 rpm2,915 - 3,085 rpm

Note

  • Since the meter display tolerance range shown in the table is an individual meter standard, it is a reference value when making vehicle inspections.
  • Confirm that the needle movement is not sluggish when gradually increasing and decreasing the engine rpm.
  • Compare with the engine rpm display and confirm that it is within the meter display tolerance range.
