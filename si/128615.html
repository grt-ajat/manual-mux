Keyless entry system check (BCM with PESS)

1. Keyless entry system check description of function

The keyless entry system functions as a vehicle entry device. The PESS control unit sends door lock and tailgate lock operation request signals to the BCM when the outside door handle is gripped, the outside door handle lock button is pressed, the tailgate unlock switch is pressed, the tailgate lock switch is pressed, or when the lock/unlock button on the electronic key is pressed. The BCM performs the requested function when signals are received from the PESS control unit with all the doors and the tailgate closed. System malfunctions may occur due to depleted electronic key or vehicle batteries, or due to electromagnetic disturbance from aftermarket devices such as power inverters or computers. Malfunctions may also occur due to electromagnetic disturbance in high-traffic areas.
The keyless entry function has the following functions.

Lock and unlock functions by lock and unlock requests from the PESS control unit
Tailgate unlock function
Answer-back function
Interior light ON/OFF interlocking function

2. Keyless entry system check

1. Diagnostic system check

Refer to "223.Description Body Control 36C.Functional Inspection(BCM with PESS) diagnostic system check-body controls".

2. DTC verification

If DTCs B0970, B2555, B3100, B3130, or B3135 are set, diagnose the applicable DTCs.

If DTCs relating to the PESS control system are set, go to the applicable DTC diagnosis.

3. Inspection of keyless entry system operation

Lock and unlock the doors with the PESS function.

If locking/unlocking is not possible, go to Electronic key programming verification.

If there is no answer-back when locking and unlocking, go to Circuit diagnosis 3.

If the dome light does not operate when unlocked, refer to Operation abnormality of the dome light function in the Description Body, Cab, Accessories section, and inspect.

Refer to "210.Description Body, Cab, Accessories 23D.Symptom(All models) operation abnormality of the dome light function".

4. Electronic key programming verification

Check whether the electronic key has been programmed with a scan tool.

If it has already been programmed, go to Inspection of PESS control unit signal circuit.

If it has not been programmed, program the electronic key information into the PESS control unit. Go to Repair verification.

Refer to "electronic key setting" in "PESS control unit setting".

Refer to "PESS control unit setting".

5. Inspection of PESS control unit signal circuit

Inspect the PESS control unit signal circuit between the BCM and the PESS control unit for an open circuit, high resistance, a short to the power supply circuit, or a short to ground. Go to Repair verification.

If the circuit is normal, go to Circuit diagnosis 1.

6. Circuit diagnosis 1

Close all doors and the tailgate.

Observe the Door Switch display on the scan tool.

If CLOSE is displayed, go to BCM harness connector inspection.

If OPEN is displayed, go to Door switch and tailgate switch inspection.

7. Door switch and tailgate switch inspection

Inspect the door switch and the tailgate switch, and repair as necessary. Go to Repair verification.

Refer to "9.Body, Cab, Accessories 9A.Lighting Systems(All models) door switch inspection".

Refer to "9.Body, Cab, Accessories 9A.Lighting Systems(All models) door switch removal".

Refer to "9.Body, Cab, Accessories 9A.Lighting Systems(All models) door switch installation".

Refer to "9.Body, Cab, Accessories 9J.Security and Locks(All models) tail gate lock removal".

Refer to "9.Body, Cab, Accessories 9J.Security and Locks(All models) tail gate lock installation".

Refer to "9.Body, Cab, Accessories 9J.Security and Locks(All models) tail gate lock inspection".

If it is normal, go to Circuit diagnosis 2.

8. Circuit diagnosis 2

Turn OFF the ignition switch.

Install the door switch and the tailgate switch.

Disconnect the BCM harness connector.

Close all doors and the tailgate.

Connect a test lamp between the door switch and tailgate switch signal circuit and the power supply circuit.

If the test lamp does not illuminate, go to BCM replacement.

If the test lamp illuminates, repair the short to ground in the door switch and tailgate switch signal circuit. Go to Repair verification.

9. Circuit diagnosis 3

Disconnect the BCM harness connector.

Connect a fused jumper wire between the answer-back control circuit and the body ground.

If the hazard lamp flashes, go to Inspection of BCM harness connector.

If the hazard lamp does not flash, repair an open circuit or high resistance in the answer-back control circuit. Go to Repair verification.

10. Inspection of BCM harness connector

Inspect the connection status of the BCM harness connector, and repair as necessary. Go to Repair verification.

If the connection status is normal, go to BCM replacement.

11. BCM replacement

Replace the BCM. Go to Repair verification.

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) BCM removal".

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) BCM installation".

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) BCM setting".

12. Repair verification

Reconnect all of the disconnected fuses and harness connectors.

Close all doors and the tailgate.

Using a remote controller key, command the doors to lock and unlock.

If it does not operate properly, go to Inspection of keyless entry system operation.
