DTC P161B (Flash Code 179) Immobilizer Wrong Response (4JK1)

1. DTC P161B description of DTC

Note

  • With PESS

The ECM communicates with the PESS control unit and performs the immobilizer function. The ECM sends a request signal to the PESS control unit. The ECM receives a response signal from the PESS control unit. Both communication signals are sent and received via CAN communication.
If the ECM receives an incorrect response signal from the PESS control unit, the DTC is set.

Note

  • With Remote Keyless Entry

The ECM communicates with the ICU to execute immobilizer function. The ECM sends a request signal to the ICU. The ECM receives a response signal from the ICU. Both communication signals are carried out via CAN communication.
If the ECM receives a wrong response signal from the ICU, the DTC is set.

2. DTC P161B condition for setting the DTC

Note

  • With PESS

When the ECM receives an incorrect immobilizer response signal from the PESS control unit

Note

  • With Remote Keyless Entry

When the ECM receives a wrong immobilizer response signal from the ICU.

3. DTC P161B action taken when the DTC sets

The ECM will not illuminate the MIL or SVS lamp.
The ECM records the operating conditions at the time the diagnostic fails. The ECM stores this information in the Failure Records.
