Body mounting installation (All models)

1. Body mounting installation

1. Install the body mounting to the frame and the body.

Tightening torque： 50 N・m { 5.1 kgf・m / 37 lb・ft } Bolt 1 in the diagram

Tightening torque： 150 N・m { 15.3 kgf・m / 111 lb・ft } Bolts 2, 3 and 4 in the diagram

Note

  • When installing the bolts of the cab mounting No.2 (RH-side), use a wrench in order to prevent the bolt heads from rotating.

2. Stopper rubber installation

1. Install the stopper rubber to the frame.

Note

  • When installing the stopper rubber to the frame, apply soapy water and install to the crossmember rear coil spring bracket of the frame.
  • Check that the stopper rubber is securely inserted into the cross member rear coil spring bracket of the frame.
