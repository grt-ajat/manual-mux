Power window timer system check (BCM with Keyless Entry)

1. Power window timer system check description of function

The power window timer function enables power windows to be opened or closed within 45 seconds after the ignition switch is turned OFF. The BCM measures 45 seconds and commands the driver side power window switch to stop the power window motor operation. If any door is opened within 45 seconds, the BCM also commands the driver side power window switch to stop the power window motor operation.

2. Power window timer system check

1. Diagnostic system check

Refer to "223.Description Body Control 36C.Functional Inspection(BCM with Keyless Entry) diagnostic system check-body controls".

2. DTC verification

If DTC B316A is set, perform diagnosis for the DTC.

3. Inspection of keyless entry

Close all doors.

Using a remote controller key, command the doors to lock and unlock.

If it does not lock or unlock, refer to Keyless entry system check, and inspect as necessary.

Refer to "223.Description Body Control 36C.Functional Inspection(BCM with Keyless Entry) keyless entry system check".

If it is normal, go to Circuit diagnosis.

4. Circuit diagnosis

Turn OFF the ignition switch.

Remove the front door power window switch.

Turn ON the ignition switch.

Measure the voltage between the power window timer signal output circuit and the body ground.

Standard： 8 V

If the reading is less than or equal to the specified value, go to Inspection for open circuit in power window timer signal output circuit.

If the reading is more than or equal to the specified value, go to Inspection of front door power window switch harness connector.

5. Inspection for open circuit in power window timer signal output circuit

Inspect for an open circuit or high resistance in the power window timer signal output circuit, and repair as necessary. Go to Repair verification.

If the circuit is normal, go to Inspection of BCM harness connector.

6. Inspection of front door power window switch harness connector

Inspect the connection status of the front door power window switch harness connector, and repair as necessary. Go to Repair verification.

If the connection status is normal, replace the front door power window switch. Go to Repair verification.

Refer to "9.Body, Cab, Accessories 9C.Electrical Control, Entertainment(All models) front door power window switch removal".

Refer to "9.Body, Cab, Accessories 9C.Electrical Control, Entertainment(All models) front door power window switch installation".

7. Inspection of BCM harness connector

Inspect the connection status of the BCM harness connector, and repair as necessary. Go to Repair verification.

If the connection status is normal, go to BCM replacement.

8. BCM replacement

Replace the BCM. Go to Repair verification.

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) BCM removal".

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) BCM installation".

Refer to "9.Body, Cab, Accessories 9E.Instrumentation, Driver Info.(All models) BCM setting".

9. Repair verification

Reconnect all of the disconnected harness connectors.

Turn ON the ignition switch.

Close all doors.

Turn OFF the ignition switch.

Verify that the power window can be operated.

If it cannot be operated, go to DTC verification.
